# 第二天

## 1. 目标 

- **词汇**：MHC、HLA、抗原呈递。
- **核心**：细胞会把蛋白碎片“挂”到表面，供 TCR 识别。
- **主题**：`MHC/HLA 是什么` 的图解入门。
- **别碰**：先别死磕 MHC-I / MHC-II 全细节。

## 2. 主线: `T-细胞` 用 `TCR` 识别 `pMHC`。

**不同类型的细胞、不同来源的蛋白碎片，会走不同的呈递路线；这和 CD4 / CD8 的分工密切相关**

+ 蛋白质 = 货物
+ 肽 `peptide` = 蛋白碎片 = 货物切下来的一小段标签
+ `MHC` = 展板
+ `pMHC` = 碎片 + 展板
+ `TCR` = T 细胞 手上的 扫描器
+ `抗原呈递` = 把碎片放展板给 T-细胞 看

## 3. `MHC` 细胞表面 的 “展示蛋白碎片的平台”。

+ 细胞内部每天都在拆蛋白、更新蛋白。
+ 蛋白碎片 叫 肽链
+ 有一部分 肽链 会被放到表面展示。

T 细胞路过，看：

+ 是不是 “正常”的？
+ 有没有 病毒味？
+ 有没有 肿瘤味？
+ 有没有 异常变化？

## 4. `HLA` = 人类版本的 `MHC`

在人类身上，MHC 通常就叫 HLA。

## 5. `抗原呈递`: 把 肽链 展示给 T细胞

`抗原` = 能被免疫系统识别的那一小段信息。

+ 细胞里有很多蛋白, 其中一些蛋白被切成小碎片
+ 这些碎片装到 MHC 上, MHC 带着碎片到细胞表面
+ T细胞 用 `TCR` 检查

## 6. `TCR` 看的是 `pMHC`

`pMHC` = `peptide` + `MHC` 这个 复合物

同样一段 peptide：

+ 换个 MHC，TCR 可能就不认
+ 换个 HLA 类型，呈递效果也可能不同

钥匙（TCR）同时适配 展板形状 + 展板上摆的碎片

## 7. `pMHC` = peptide-MHC complex

+ p = peptide，蛋白碎片
+ MHC = 展板

挂着某段蛋白碎片的 MHC 复合物

“这个 TCR 识别某个 pMHC”
“这个肿瘤新抗原 pMHC 很好”
“做 pMHC 靶向”

`细胞表面有个支架，上面托着一小段碎片，TCR 上来检查`。

## 8. `CD4` 和 `CD8`

+ 都用 TCR 看 pMHC，只是偏向的工作分工不同。
+ `CD8` = 发现并杀死异常细胞
+ `CD4` = 协调免疫反应、帮助 B 细胞

# 第二个月：计算结构生物学

## 第 1 周：看结构: PDB-结构图

用 可视化工具 打开 结构图 讲清楚 “谁是谁”

- 可视化工具: 开结构、看链、看复合物、看谁和谁接触
- 蛋白链, 配体, 抗体重链轻链
- pMHC 里 peptide、MHC、TCR 分别在哪
- CD70 表面蛋白 是 什么构型

## 第 2 周：看界面

工具: 界面分析 (辅助阅读，不是自动打分)

- 标哪些界面残基
- 辅助看氢键、疏水接触、盐桥、clash
- 辅助回答: 界面为什么成立

## 第 3 周：突变逻辑

工具: 突变分析

目标不是一上来做大规模扫描，而是：

- 选 1–2 个位点
- 看小残基换大残基会不会撞
- 看极性/疏水变化会不会不合环境
- 看 interface residue 和 core residue 改动风险差别

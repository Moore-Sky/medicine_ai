# 第四周：工程概述

# 第16天：药物识别 四要素: 识别头 / 靶点 / 表位 / 激活模块

问题：

- 药物靠什么识别？
- 识别的是整个蛋白，还是一个表位？
- 识别以后怎么触发细胞反应？
- 哪些部分能计算辅助，哪些必须实验验证？

CD19-CAR-T：

- 识别头 = FMC63 scFv
- 靶点 = CD19
- 表位 = CD19 胞外结构上的一小块 3D 区域
- 激活 = CD3ζ + CD28 / 4-1BB

pMHC / CoStAR：

- 识别头 = TCR-like / scFv-like 识别模块
- 靶点 = peptide-HLA
- 表位 = 肽段 + HLA 共同形成的复合表面
- 激活 = TCR/CD3 + 共刺激模块

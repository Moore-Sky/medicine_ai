## 1. 序列 = 蛋白质的一维字符串。

蛋白质是由氨基酸连起来的。每个氨基酸可以用一个字母表示 MKWVTFISLLFLFSSAYSRGVFRR...

它不是最终形态。蛋白真正工作时，会折叠成三维结构。

序列 = 一维; 结构 = 三维; **功能 = 三维结构 + 环境 + 相互作用**

`UniProt`: 蛋白质百科全书。

| 看什么                 | 意义                               |
| ---------------------- | ---------------------------------- |
| 蛋白全名               | 这个蛋白到底叫什么                 |
| 序列                   | 氨基酸一维字符串                   |
| 长度                   | 有多少个氨基酸                     |
| 结构域                 | 哪一段负责什么功能                 |
| 胞外 / 跨膜 / 胞内区域 | 哪部分暴露在细胞外，药物能不能摸到 |
| 功能说明               | 它在细胞里大概干嘛                 |
| 相关疾病               | 和哪些病有关                       |
| 交叉引用               | 有没有 PDB 结构、论文、数据库记录  |

有用：

- 这个蛋白有没有胞外段？
- 这个胞外段是哪一段？
- 药物理论上能不能接触到？
- 有没有已知结构？

例子: CD19

- CD19 哪一段在 B 细胞表面外面？
- FMC63 识别的是 CD19 胞外结构上的哪一块？

`BLAST` ≈ 我有一段序列，世界上有没有类似的蛋白？

| 问题                       | BLAST 能干嘛           |
| -------------------------- | ---------------------- |
| 这个序列像哪个蛋白？       | 找相似蛋白             |
| 人和小鼠的 CD19 像不像？   | 比较同源性             |
| 某个区域是否保守？         | 看不同物种是否都差不多 |
| 抗体识别区域是否容易变化？ | 看表位附近是否保守     |

## 2. 结构

- 结构，首先指三维空间结构。
- 亲疏水、氢键、静电、盐桥、范德华力这些，是在三维结构上进一步分析出来的相互作用。

#### 2.1 第一层：三维空间结构

蛋白不是一根直线，而是会折叠成： α 螺旋 / β 折叠 / 环区 / 结构域 / 表面凹槽 / 凸起区域 / 口袋 / 柔性区域

在 `PyMOL` / `ChimeraX` 看到的球棒、丝带、表面图，主要就是这个。

问题:

- 这个蛋白长什么样？
- 哪部分在 外面 / 里面？
- 抗体从哪个方向贴上去？

#### 2.2 第二层：界面分析 -- 局部接触关系

- 两个分子结合时，不是“整个蛋白贴整个蛋白”。
- A 分子表面的一小块 接触 B 分子表面的一小块

比如：

FMC63 scFv的CDR区 接触 CD19胞外结构上的某个表位区域

问题:

- 哪些残基靠得很近？
- 哪些残基真的参与接触？
- 接触面积多大？
- 有没有关键接触点？

#### 2.3 第三层：物理化学作用

这不是结构，而是：在三维结构基础上，对接触关系的物理解释。

- 结构 = 空间坐标
- 界面 = 两个结构接触的区域
- 相互作用 = 界面上为什么能粘住的原因

| 类型       | 简单理解                 |
| ---------- | ------------------------ |
| `氢键`     | 像局部小钩子             |
| `静电`吸引 | 正负电荷互相吸           |
| `盐桥`     | 比较强的正负电荷配对     |
| `疏水`接触 | 不喜欢水的区域抱团       |
| `范德华力` | 很近距离的小吸引         |
| `形状互补` | 凸的地方正好插进凹的地方 |

#### 2.4 工具 `PyMOL` / `ChimeraX`

操作: 打开结构, 旋转结构, 选中某条链, 显示表面, 看残基, 看距离, 看界面, 截图画图

`ChimeraX`: 更现代，对新手友好

- 优点：界面舒服, 可视化清楚, 适合初学看结构, 处理大结构比较方便
- 缺点：一些老教程、老论文图更偏 PyMOL

`PyMOL`: 偏经典，论文图很多用它做。

- 优点：脚本化强, 出图漂亮, 资料多, 结构生物学领域很常见
- 缺点：对新手不算最友好, 刚开始容易陷入命令细节

建议:

- 先用网页看 PDB 结构
- 真要本地看，优先 `ChimeraX`
- 以后要做规范图和脚本，再补 `PyMOL`

## 3. 界面 = 两个分子接触的表面区域。

- 抗体这边：第 31、52、99 位残基参与接触
- 抗原那边：第 85、87、112 位残基参与接触

#### 3.1 工具 `PDBePISA` (网页版)

- 两个链之间有没有稳定界面？
- 接触面积多大？
- 哪些残基参与接触？
- 可能有哪些氢键/盐桥？
- 这个复合物是不是合理的生物组装？

## 4. 例子: `CD19` / `FMC63`

- 靶点 = 被调用的服务
- 表位 = 服务暴露出来的 API surface
- 识别头 = 调用方的 client
- 界面 = client 和 API 真正发生交互的接触面
- 亲和力 = 两边粘得稳不稳
- 特异性 = 会不会误调用别的服务

## 5. 其他工具

#### 5.1 `RCSB PDB` = 蛋白三维结构数据库

结构生物学界的 GitHub / npm registry。

| 看什么               | 意义                       |
| -------------------- | -------------------------- |
| 有没有这个蛋白的结构 | 有现成的就别急着预测       |
| 是单体还是复合物     | 有没有抗体/配体一起        |
| 分辨率               | 结构可靠程度之一           |
| chain                | 哪条链是抗原，哪条链是抗体 |
| ligand               | 有没有小分子、肽段、离子   |
| download             | 下载 PDB/mmCIF 文件        |

#### 5.2 看突变影响 `FoldX` / `Rosetta`

做啥: 把某个氨基酸从 A 改成 B，会不会影响稳定性/结合？

- CD19 某个残基突变，会不会影响 FMC63 结合？
- FMC63 某个 CDR 残基突变，会不会增强/减弱结合？

比较:

- `FoldX`: 轻量，常用于估算突变对稳定性、结合能的影响。
- `Rosetta`: 更大、更复杂，是蛋白建模、设计、打分的一整套工具体系。

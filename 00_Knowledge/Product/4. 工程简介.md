# 第四周：工程概述

# 1. 第16天: 结构基础、亲和力优化、干湿结合

- **核心**：入口是“先理解如何识别”，再谈优化。
- **看什么**：抗体亲和力成熟/优化入门综述。

产出一个《识别头 / 靶点 / 尾巴 / 激活》四要素图

# 2. 第17天: 为何先学结构，不先堆环境

- **核心**：早期先理解方法和可复现性，不急于全量安装。
- **做什么**：整理软件清单（先列不装）：
  - 结构可视化工具。
  - 论文阅读工具。
  - 蛋白结构预测/分析工具。
  - 序列处理工具。

整理软件清单时，建议明确输出格式：

- 软件名
- 用途
- 我为什么可能需要它
- 现在先不装的原因

# 3. 第18天: 算力、租用、需求优先

- **核心**：先明确需求，再决定是否花钱租算力。
- **做什么**：写一页《如果以后跑结构，我最关心的算力问题》。
  - 我以后可能跑什么任务？
  - 这些任务是结构预测、对接还是可视化？
  - 哪些任务本地可做，哪些可能要租？
  - 什么情况下才值得花钱？

# 4. 第19天: 综述阅读法、论文阅读法

- **核心**：阅读顺序应为：综述 -> 图解/讲解 -> 经典案例 -> 论文。
- **做什么**：整理一个“读论文模板”。

# 5. 第20天: 总结

1. 写一页《我现在怎么理解这个项目》。
2. 写一页《我现在还不懂什么》。
3. 写一页《下个月该做什么》。

建议覆盖：

- CD70 路线是什么。
- pMHC / CoStAR 路线是什么。
- 为什么先学 CD19/CD70 再碰 pMHC。
- 本月最重要的 10 个词。
- 当前主要疑问（如表位定义、亲和力与特异性平衡、连接方式与激活关系、CDR 优化路径）。
- 下月行动项（固定综述类型、选一个结构案例、学一个可视化工具、再决定是否上算力环境）。

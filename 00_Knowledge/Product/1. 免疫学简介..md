# 第一周: 免疫学简介

# 第1天: 抗原、免疫、T 细胞、B 细胞、白细胞

[可汗学院 - 人类生物学 - 免疫学 部分](https://zh.khanacademy.org/science/biology/human-biology)

## 1. 通俗解释

- 免疫系统 = 城市安保系统。
- 白细胞 = 安保人员总称，不是一种细胞。
- T 细胞和 B 细胞是相对“可编程”的岗位，每个细胞带不同识别头，能识别不同目标。
- 抗原像“敌人身上的标志”，抗体像“专门去结合这个标志的工具”。
- 免疫系统通常不是识别整个病毒，而是识别其中很小的一段，叫`表位`（epitope）。
- T 细胞更像“看证件办事”：它识别的是细胞表面展示出来的`抗原片段 + MHC`（即`pMHC`）。
- B 细胞更像“直接看脸”：可以直接结合天然抗原（比如蛋白表面结构）。

## 2. 功能

免疫系统的核心功能可以概括为“保安 + 消防 + 清洁 + 维修”：

- 识别和清除外来病原体（细菌、病毒、真菌等）。
- 清理体内异常成分（老化细胞、受损细胞、突变细胞）。
- 在损伤后启动修复（止血、清理、重建）。

常见细胞分工：

- T 细胞
  - 杀伤性 T 细胞：杀死被感染细胞或部分肿瘤细胞。
  - 辅助性 T 细胞：协调其他免疫细胞，帮助 B 细胞应答。
- B 细胞
  - 识别抗原后分化为浆细胞，产生抗体。
- 巨噬细胞
  - 吞噬病原和碎片、清理战场、释放信号、并可做抗原呈递。

## 3. 结构

免疫系统 = 细胞层 + 信号层 + 器官层

- 细胞层（执行者）
  - T 细胞、B 细胞、NK 细胞、巨噬细胞、中性粒细胞、树突状细胞等。
- 信号层（通信网络）
  - 细胞因子、趋化因子、抗体等，负责“通知、调度、放大反应”。
- 器官层（训练与集结）
  - 骨髓：血细胞来源。
  - 胸腺：T 细胞成熟。
  - 淋巴结：识别与激活的重要场所。
  - 脾脏：过滤血液与处理血中抗原。
  - 扁桃体、肠道相关淋巴组织：边界防御站点。

关键名词关系：

- 抗原（antigen）：可被免疫系统识别的分子或片段。
- 抗体（antibody）：B 细胞系统产生的结合工具。
- 抗原呈递：把抗原切成片段并展示在 MHC 上给 T 细胞看。
- MHC：细胞表面的“展示托盘”，T 细胞通过它读取信息。

## 4. 过程

### 4.1 天然免疫与适应性免疫协作

- 先天免疫（如巨噬细胞、中性粒细胞）先快速响应，模式识别为主。
- 适应性免疫（T/B 细胞）随后进入精确识别和记忆阶段。

### 4.2 T/B 细胞“可编程”逻辑

- 体内先产生大量带不同受体的 T/B 细胞（海量“钥匙库”）。
- 巡逻时不断与目标接触并试探匹配。
- 一旦命中目标，对应克隆扩增，形成更强应答与免疫记忆。

### 4.3 T 细胞识别流程（核心）

1. 抗原被摄取并切割成短肽。
2. 短肽装载到 MHC 分子上形成`pMHC`。
3. 抗原呈递细胞把`pMHC`展示给 T 细胞。
4. TCR 识别到匹配`pMHC`后，T 细胞激活、扩增、执行效应功能。

### 4.4 人工干预流程（ACT/工程化细胞）

- 检索扩增路线（ACT）：从体内找到少量能识别目标的 T 细胞，分离、体外扩增、回输。
- 设计改造路线：直接给细胞装新识别模块或改内部逻辑，再扩增回输。
  - 代表方法：`TIL`、`TCR-T`、`CAR-T`。

## 5. 作用

### 5.1 生理作用

- 防感染、防扩散。
- 监视并清除部分异常细胞（含肿瘤相关异常）。
- 参与组织修复与稳态维持。

### 5.2 炎症的双刃剑作用

- 急性炎症：短期、有效、可收尾，通常有利于清除威胁。
- 慢性炎症：长期低度激活，可能造成组织损伤并增加癌症风险。

常见关联例子：

- 慢性乙肝/丙肝与肝癌风险升高。
- 幽门螺杆菌相关慢性胃炎与胃癌风险升高。
- 慢性肠炎与结直肠癌风险升高。
- 吸烟、肥胖相关慢性炎症与多种肿瘤风险升高。

# 第2天: MHC、HLA、抗原呈递

## 1. 目标

- **词汇**：MHC、HLA、抗原呈递。
- **核心**：细胞会把蛋白碎片“挂”到表面，供 TCR 识别。
- **主题**：`MHC/HLA 是什么` 的图解入门。
- **别碰**：先别死磕 MHC-I / MHC-II 全细节。

## 2. 主线: `T-细胞` 用 `TCR` 识别 `pMHC`。

**不同类型的细胞、不同来源的蛋白碎片，会走不同的呈递路线；这和 CD4 / CD8 的分工密切相关**

- 蛋白质 = 货物
- 肽 `peptide` = 蛋白碎片 = 货物切下来的一小段标签
- `MHC` = 展板
- `pMHC` = 碎片 + 展板
- `TCR` = T 细胞 手上的 扫描器
- `抗原呈递` = 把碎片放展板给 T-细胞 看

## 3. `MHC` 细胞表面 的 “展示蛋白碎片的平台”。

- 细胞内部每天都在拆蛋白、更新蛋白。
- 蛋白碎片 叫 肽链
- 有一部分 肽链 会被放到表面展示。

T 细胞路过，看：

- 是不是 “正常”的？
- 有没有 病毒味？
- 有没有 肿瘤味？
- 有没有 异常变化？

## 4. `HLA` = 人类版本的 `MHC`

在人类身上，MHC 通常就叫 HLA。

## 5. `抗原呈递`: 把 肽链 展示给 T细胞

`抗原` = 能被免疫系统识别的那一小段信息。

- 细胞里有很多蛋白, 其中一些蛋白被切成小碎片
- 这些碎片装到 MHC 上, MHC 带着碎片到细胞表面
- T细胞 用 `TCR` 检查

## 6. `TCR` 看的是 `pMHC`

`pMHC` = `peptide` + `MHC` 这个 复合物

同样一段 peptide：

- 换个 MHC，TCR 可能就不认
- 换个 HLA 类型，呈递效果也可能不同

钥匙（TCR）同时适配 展板形状 + 展板上摆的碎片

## 7. `pMHC` = peptide-MHC complex

- p = peptide，蛋白碎片
- MHC = 展板

挂着某段蛋白碎片的 MHC 复合物

“这个 TCR 识别某个 pMHC”
“这个肿瘤新抗原 pMHC 很好”
“做 pMHC 靶向”

`细胞表面有个支架，上面托着一小段碎片，TCR 上来检查`。

## 8. `CD4` 和 `CD8`

- 都用 TCR 看 pMHC，只是偏向的工作分工不同。
- `CD8` = 发现并杀死异常细胞
- `CD4` = 协调免疫反应、帮助 B 细胞

# 第3天: pMHC、肽段、突变抗原

## 1. 目标

- **术语**：pMHC、肽段、突变抗原。
- **核心**：
  - pMHC = 被 MHC 展示在细胞表面的一个短肽；
  - 胞内突变本身看不见，切成肽后，能以 pMHC 的形式“露头”，于是被 T 细胞发现

## 2. 术语

1. 肽段 `peptide`

蛋白质片段。T 细胞通常不是直接看整条蛋白，而是看这些被展示出来的小片段。

2. `pMHC`（peptide–MHC complex）

不是“肽”单独摆在外面，而是 `肽 + MHC 分子` 的复合物，挂到细胞表面。

`TCR` 识别的也不是裸肽，而是整个 pMHC。

3. `突变抗原` neoantigen

肿瘤细胞 突变 产生正常细胞 没有的 异常肽。

异常肽 装到 MHC 展示，成 neoantigen pMHC，是癌症免疫治疗特别重视的目标。

## 3. 主线

- 突变要变成“有用的 neoantigen”，至少要过几关:
- 突变蛋白 → 切成短肽 → 少数装进 MHC → 形成 pMHC 展示 → 被 TCR 识别，触发 T 细胞反应
- 上面的每一关的损耗很大

文章:

主读 1：`Neoantigens: promising targets for cancer therapy`（2023 review）

把 neoantigen / pMHC 的来源、为什么重要、怎么走到疫苗/TCR-T/联合治疗，整体串得比较完整; 建立“突变 → pMHC → T 细胞 → 治疗”的总图。

主读 2：`MHC-II neoantigens shape tumour immunity and response to immunotherapy`（2019）

打破初学者误区：不是只有 MHC-I/CD8 重要，MHC-II / CD4 路线在抗肿瘤免疫里也可能是关键角色。对你后面理解“为什么免疫系统不是单兵作战”很有帮助。

加餐：`Structural basis for T cell recognition of cancer neoantigens`（2023 review）

更偏 `结构生物学`。今天不必细读，但适合你先翻图感受一件事：TCR 识别的不是抽象“名字”，而是一个很具体的三维表面几何。这个直觉对你以后学 pMHC 设计很重要

## 4. 动态性

并不是每个肿瘤异常都能出现在细胞表面成为 pMHC。

对某个具体病人 A 来说，他肿瘤里真正能被展示出来的 neoantigen 往往只是全部异常中的一小部分。

不同肿瘤细胞亚群、微环境、时间点，pMHC 可能不完全一样，有些靶点稳定存在，有些靶点则会增减或消失

1. “50 个异常指标”不等于“50 个 pMHC 靶点”

如果你说的是“某类未分化胃癌研究里发现过 50 个异常分子/异常突变/候选标记”，那到病人 A 身上，先要打一个大折扣：

- 他未必真的有这 50 个；
- 有了也未必都表达；
- 表达了也未必都能被切成合适肽；
- 切出来也未必适配他自己的 HLA；
- 适配了也未必足够稳定、足够多，能让 T 细胞看见

2. 一个细胞表面通常不是只有一个 pMHC，而是一大堆不同 pMHC 的混合谱

- T 细胞看到的不是“这个细胞今天只挂 A 标签，明天只挂 B 标签”
- 细胞表面同时挂着很多 pMHC，其中绝大多数是普通自身肽，少数可能是肿瘤相关或突变来源肽。
- 问题：`那些真正有价值的 neoantigen pMHC 往往数量少、密度低，而且还可能被肿瘤逃逸机制进一步压低`。

3. 会“变”，但不是纯随机闪烁

- 肿瘤有异质性，不同细胞亚群本来就不一样；
- 微环境 / 治疗压力 改展示谱；
- 肿瘤 可能通过丢失 neoantigen、降低 MHC-I、损坏 TAP/加工通路 来 逃避免疫。
- 更像: 一套会漂移、会重排、还会被肿瘤主动作弊的展示菜单，而不是完全随机抽卡

## 5. 治疗难题

不是“永远杀不死”，而是 `单靶点、单回合、一次性清零”这件事在异质性肿瘤里很难`

- `肿瘤异质性`：不同癌细胞挂出来的 pMHC 不一样。
- `免疫逃逸`：被打疼以后，肿瘤会丢靶点、降 HLA、坏掉抗原加工通路，躲开 T 细胞。

治疗思路:

- 不是 “穷举全集”，而是 “优先打最关键、最稳定、最像树干的那部分”，
- 尽量选 `clonal neoantigens`: 多数 肿瘤细胞共享的突变来源靶点。
- 临床和转化研究里，clonal neoantigen burden 与更好的免疫治疗反应相关，而纯 subclonal 靶点更容易漏网

`表位扩展` (epitope spreading) : 先打掉一部分肿瘤细胞，细胞死亡会释放更多肿瘤抗原，经过树突细胞处理后，可能进一步诱导对其他肿瘤抗原的 T 细胞反应

`旁观者杀伤` (bystander killing): T-细胞 先被“抗原阳性”的肿瘤细胞激活后，在一定条件下，能连带杀伤附近一些“抗原阴性/抗原低”的肿瘤细胞。

## 6. `HLA 限制性`: 为什么同一个突变，在病人甲可以成为靶点，在病人乙却根本不行？

能不能打 = 突变 × 病人HLA型别 × 能否稳定展示 × 有没有合适TCR识别

- TCR 识别的是“肽 + 特定 HLA”的组合，不是肽单独一个。
- 不同人的 HLA 不同，所以同一个突变肽在甲能展示，在乙可能根本装不上去。
- `pMHC 靶向治疗通常不是只看“有没有这个突变”，而是看 “有没有这个突变 + 有没有对应 HLA”`。

类比:

- 突变 = 原材料
- HLA = 展示架 / 插槽
- pMHC = 摆出来的成品
- TCR = 检票员

同一个“原材料”，换一个插槽，摆出来的成品就可能完全不是一回事。

所以病人甲能成靶点，病人乙不行，很多时候不是因为突变不同，而是因为HLA 背景不同

# 第4天: TCR、BCR、抗体

## 1. 目标

- BCR 是什么，和抗体是什么关系
- 抗体怎么直接认表面目标
- TCR 为什么不能像抗体一样随便抓表面蛋白

## 2. 主线: 免疫系统识别路线

### 2.1. TCR: 查证件

TCR: T 细胞 表面的受体, 用于 识别 pMHC

- 细胞先把体内蛋白切成小片段, 用 MHC/HLA 挂到表面
- TCR 检查 “展示牌”

`细胞内部异常` 路线:

- `TCR-T` / `pMHC 靶向` / `新抗原识别`
- 异常蛋白先被切片，再通过 pMHC 展示出来，再让 TCR 去识别。

### 2.2. BCR: 看长相

直接识别 天然状态下 暴露出来 的 抗原结构。

- 病毒表面某个蛋白
- 细胞表面某个受体
- 肿瘤细胞表面的某个分子

`BCR`: B-细胞 表面的受体, 让 B-细胞 先识别目标。

识别之后，B-细胞被激活、分化，最终可以大量分泌抗体。

- BCR：B-细胞表面的识别器
- 抗体：B-细胞 成熟后 分泌出去的武器

`表面靶点` 路线:

- 抗体药 / `CAR-T` / `ADC`
- `CD19` / `CD20` / `HER2` / `CD70`

# 3. 抗体怎么直接认表面目标

### 3.1. `抗体`

`抗体`: 可溶性的蛋白武器，能在体液里到处跑。

直接结合目标表面的某个结构, 作用:

- `阻断`: 堵住某个受体，不让它继续传信号。
- `中和`: 把病毒关键部位占住，让它进不了细胞。
- `标记`: 通知 免疫细胞 这个东西该处理了
- `招募杀伤`: 通过 Fc-段 招募 NK-细胞、巨噬细胞 清除目标。

### 3.2. 怎么认

`抗原表位`（epitope）, 目标表面上的一小块可被认出的局部形状

- 抗体: 锁 / 插头
- 目标的`抗原表位`: 钥匙 / 插座
- 匹配 三维结构: **大体匹配 + 局部微调 + 多种弱相互作用叠加**

### 3.3. 为啥能认

目标表面的某些分子，本来就暴露在外面;

抗体在体液里流动时，碰到以后就可以直接结合。

- 病毒外壳蛋白
- 细菌表面分子
- 肿瘤细胞表面的受体
- 细胞膜上的 CD19、HER2、EGFR 之类分子

### 3.4. BCR & 抗体

- BCR 可看成“长在 B 细胞膜上的抗体”；
- 抗体 可以看成 “分泌出去、游离在体液里的 BCR 同款识别头”。

二者识别的靶标类型很接近, 区别在于：

- BCR 挂在 B 细胞表面，用来“感知并启动 B 细胞”
- 抗体 被分泌出去，用来 “在体内到处结合目标并发挥效应”

# 4. TCR 为什么不能像抗体一样随便抓表面蛋白

很多危险，不在细胞表面，而在细胞里面：

- 病毒在细胞内复制出来的蛋白
- 肿瘤细胞内部产生的突变蛋白
- 异常表达的内部蛋白片段

TCR 的任务: 看细胞正在展示什么内部样本。

T 细胞激活, 连着整套很严格的激活机制：

- CD3 复合体传信号
- CD4 / CD8 共受体参与
- 共刺激分子参与
- 机械力与免疫突触形成
- 激活阈值控制

# 第5天: 自身耐受、阴性筛选、为何攻击病毒/肿瘤

# 1. 目标

免疫系统不是“认识所有敌人”，而是先学会“不要乱打自己”，再对那些呈现异常信号的细胞提高警惕。

- 免疫系统为什么平时不攻击自己？
- T 细胞明明是随机生成受体，为什么不会到处误伤？
- 病毒感染细胞、肿瘤细胞，为什么更容易被盯上？
- “自身” 和 “异常” 的边界到底是怎么划出来的？

### 总体理解: 免疫系统 = 高覆盖 + 多层风控

- 随机生成识别器: 先尽量做出多样性, 才有机会覆盖未知敌人
- 阴性筛选: 太容易打自己的，先删掉
- 外周耐受: 漏网之鱼再加刹车, 防止轻易自体攻击

`阴性筛选` = T 细胞出厂前的“大筛查”
`自身耐受` = 系统“不攻击自己”的总原则
`中央耐受` = 在训练营里处理一批危险细胞
`外周耐受` = 放到身体里后继续加保险
`异常信号` = 为什么感染/肿瘤会更容易被注意到

# 2. T-细胞 构造: 大量随机生成，再把太危险的删掉。

## 2.1 T-细胞受体 是 “随机生成”的

- T-细胞的 TCR 不是提前为某个病毒专门设计好的, 而是 随机产生 很多 不同形状的“识别头”
- 随机生成，就很可能出现“能认自己”的 TCR; 如果不处理，免疫系统会自残

## 2.2 `阴性筛选`：先把“太容易打自己”的 T 细胞淘汰

- `胸腺`: T-细胞 的 重要训练场
- T-细胞 接触 “自己人的分子片段”, 如对自身片段反应太强, 它就会被诱导死亡

# 3. 免疫系统 不是 “绝对区分自己和敌人”

- 底线: 不要强烈攻击自身
- 结合环境，判断 当前 有没有危险
- 识别: 是否 异常、危险、该启动反应

注意: 肿瘤细胞是“自己变坏了”, 不完全是外来细胞

# 4. 为什么 病毒感染细胞 更容易被打

病毒感染细胞，比较像“有明显外来成分”。

- 病毒进入细胞
- 利用 细胞机器 生产 病毒蛋白
- 病毒蛋白 会被切成小片段, 通过 MHC 挂到表面
- T 细胞看到“这不是正常人体平时该挂的内容”, 就更可能发动攻击

# 5. 为什么肿瘤细胞也可能被打，但更麻烦

肿瘤细胞：人体细胞突变, 它挂出来的很多东西大部分像“自己人”。

## 5.1 突变产生新肽段

突变抗原 对应的 pMHC, 某些 T 细胞可能识别到它。

## 5.2 表达模式异常: 虽然还是自己人，但状态不对劲

肿瘤细胞 可能：某些蛋白表达过高, 在 错误时间/错误位置 表达, 呈现 压力信号、炎症 环境变化

# 6. 阴性筛选很重要，但不可能完美。

- 人体自身分子太多, 场景太复杂, 有些抗原在胸腺里不一定完整呈现
- 有些 T 细胞对自身只是“中等反应”，不一定被删掉

第二层保险：`外周耐受`: 离开胸腺、进入全身以后，还要继续防止误伤。

- 没有足够激活信号，就不完全启动
- 某些细胞 进入 无反应状态
- 某些抑制机制把 过激反应压住
- 负责“踩刹车”的调节细胞

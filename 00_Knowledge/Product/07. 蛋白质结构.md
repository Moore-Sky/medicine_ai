# 第7周 蛋白质结构基础：从氨基酸到界面

氨基酸 → 残基 → 序列 → 二级结构 → 三级结构 → 四级结构 / 复合物 → 界面 → 亲和力 / 特异性 / 功能

# 第31天 氨基酸 和 残基

知道 21 种氨基酸不是重点背名字，而是按性质分组。

| 类型   | 例子               | 对结构的意义               |
| ------ | ------------------ | -------------------------- |
| `疏水` | Leu, Ile, Val, Phe | 常埋在内部，也参与疏水界面 |
| `极性` | Ser, Thr, Asn, Gln | 容易形成氢键               |
| `正电` | Lys, Arg, His      | 可参与静电 / 盐桥          |
| `负电` | Asp, Glu           | 可参与静电 / 盐桥          |
| `特殊` | Gly, Pro, Cys      | 影响柔性、转角、二硫键     |

# 第34天 四级结构：多条链怎么组成复合物

理解 chain、complex、subunit、antibody heavy / light chain

用 CD19/FMC63 对应：

- CD19 = 一条抗原链
- FMC63 heavy chain = 抗体重链
- FMC63 light chain = 抗体轻链
- 三者组合形成复合物

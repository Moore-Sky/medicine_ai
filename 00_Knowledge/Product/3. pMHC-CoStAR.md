# 第三周：pMHC / CoStAR

# 1. 第11天: 胞内抗原、为何普通 CAR-T 够不着

- **核心**：胞内靶点不像 CD70 那样直接暴露在表面。
- **看什么**：为什么胞内靶点难做抗体药/CAR-T。

# 2. 第12天: p53/TP53、突变短肽、表面呈递

- **核心**：识别目标通常是“突变肽段 + MHC”，不是整段 p53。
- **别碰**：先不看完整肿瘤测序流程。

# 3. 第13天: 天然 TCR 的限制

- **核心**：结构预测尚难替代实验验证，且脱靶风险高。
- **看什么**：TCR specificity / cross-reactivity 科普或综述摘要。

# 4. 第14天: CoStAR

- **核心**：可先粗略理解为：将更高亲和力的识别模块与 TCR 信号底盘结合，以增强对低密度 pMHC 靶点的识别与激活。
- **看什么**：CoStAR 论文图 1、摘要、图解。

# 5. 第15天: 亲和力、特异性、脱靶

- **核心**：三角矛盾是“更强识别、更强点火、可控脱靶”。
- **输出**：写一页《为什么 pMHC/CoStAR 比 CD70 更难》。

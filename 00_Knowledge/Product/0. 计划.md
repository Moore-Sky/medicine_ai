# 程序员转入免疫治疗项目：4 周入门路线

建立 免疫项目 的 基础认知，理解 问题和路径，再决定 工具与环境。

# 1. 验收标准

- 能用自己的话解释 pMHC。
- 能解释 CD70 与 pMHC 是两条不同路线。
- 能讲清楚 CD19 为什么是经典案例。
- 能说明 CoStAR 试图解决的问题。
- 理解 “先抓认知，不急着上环境” 的原因。

# 2. 路线: 每天一小时

- 第 1 周：免疫系统在干嘛
- 第 2 周：表面靶点: CD19 / CD70
- 第 3 周：pMHC / Co-StAR
- 第 4 周：明确工程位置
- 第 5 周: 通过 CD19 看结构
- 第 6 周:通过 CD19 看界面

# 3. 别碰什么

- 实验方案：当前 重点是认知，不是实战。
- 环境搭建：别折腾 GPU、Docker、生信全家桶。
- 把AF当万能：结构预测不能替代关键结合与功能验证。
- 讨论商业化AF3方案：现阶段, 性价比不高。
- 追新闻：**应该优先看综述, 经典案例**。
- 一次性读懂原始论文。应该先 **摘要、结论、综述文、图解**，再逐步回头补方法学。

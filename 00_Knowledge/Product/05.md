# 第5天: 自身耐受、阴性筛选、为何攻击病毒/肿瘤

# 1. 目标

免疫系统不是“认识所有敌人”，而是先学会“不要乱打自己”，再对那些呈现异常信号的细胞提高警惕。

+ 免疫系统为什么平时不攻击自己？
+ T 细胞明明是随机生成受体，为什么不会到处误伤？
+ 病毒感染细胞、肿瘤细胞，为什么更容易被盯上？
+ “自身” 和 “异常” 的边界到底是怎么划出来的？

### 总体理解: 免疫系统 = 高覆盖 + 多层风控

+ 随机生成识别器: 先尽量做出多样性, 才有机会覆盖未知敌人
+ 阴性筛选: 太容易打自己的，先删掉
+ 外周耐受: 漏网之鱼再加刹车, 防止轻易自体攻击

`阴性筛选` = T 细胞出厂前的“大筛查”
`自身耐受` = 系统“不攻击自己”的总原则
`中央耐受` = 在训练营里处理一批危险细胞
`外周耐受` = 放到身体里后继续加保险
`异常信号` = 为什么感染/肿瘤会更容易被注意到

# 2. T-细胞 构造: 大量随机生成，再把太危险的删掉。

## 2.1 T-细胞受体 是 “随机生成”的

+ T-细胞的 TCR 不是提前为某个病毒专门设计好的, 而是 随机产生 很多 不同形状的“识别头”
+ 随机生成，就很可能出现“能认自己”的 TCR; 如果不处理，免疫系统会自残

## 2.2 `阴性筛选`：先把“太容易打自己”的 T 细胞淘汰

+ `胸腺`: T-细胞 的 重要训练场
+ T-细胞 接触 “自己人的分子片段”, 如对自身片段反应太强, 它就会被诱导死亡

# 3. 免疫系统 不是 “绝对区分自己和敌人”

+ 底线: 不要强烈攻击自身
+ 结合环境，判断 当前 有没有危险
+ 识别: 是否 异常、危险、该启动反应

注意: 肿瘤细胞是“自己变坏了”, 不完全是外来细胞

# 4. 为什么 病毒感染细胞 更容易被打

病毒感染细胞，比较像“有明显外来成分”。

+ 病毒进入细胞
+ 利用 细胞机器 生产 病毒蛋白
+ 病毒蛋白 会被切成小片段, 通过 MHC 挂到表面
+ T 细胞看到“这不是正常人体平时该挂的内容”, 就更可能发动攻击

# 5. 为什么肿瘤细胞也可能被打，但更麻烦

肿瘤细胞：人体细胞突变, 它挂出来的很多东西大部分像“自己人”。

## 5.1 突变产生新肽段

突变抗原 对应的 pMHC, 某些 T 细胞可能识别到它。

## 5.2 表达模式异常: 虽然还是自己人，但状态不对劲

肿瘤细胞 可能：某些蛋白表达过高, 在 错误时间/错误位置 表达, 呈现 压力信号、炎症 环境变化

# 6. 阴性筛选很重要，但不可能完美。

+ 人体自身分子太多, 场景太复杂, 有些抗原在胸腺里不一定完整呈现
+ 有些 T 细胞对自身只是“中等反应”，不一定被删掉

第二层保险：`外周耐受`: 离开胸腺、进入全身以后，还要继续防止误伤。

+ 没有足够激活信号，就不完全启动
+ 某些细胞 进入 无反应状态
+ 某些抑制机制把 过激反应压住
+ 负责“踩刹车”的调节细胞

# Day 1 — 人体层级总览

用自己的话说出：**人体不是零件箱，而是一个多层级协作网络**。

细胞 → 组织 → 器官 → 系统 → 人体

人体: 分层组织的系统，不是一堆器官的拼盘。

- **词汇**：细胞 --> 组织 --> 器官 --> 系统
- **结构**：
    - 为什么说人体是“层层嵌套”的？ 
    - 例如: 心脏属于器官，心肌属于组织，心肌细胞属于细胞，这条层级链怎么串起来？
- **功能**：
    - 为什么人体不是让每个器官各干各的，而是靠不同层级共同协作？
- **过程**：
    - 遇到一个全身性问题，比如 缺氧、发烧、脱水，系统层的变化会怎样一步步影响到器官、组织和细胞？

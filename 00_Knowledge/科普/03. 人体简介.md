# 第1周: 人体简介

# 第1天 人体: 细胞-组织-器官

## 1. 概述

- **细胞 → 组织 → 器官 → 系统 → 人体**
- **通常多个相似细胞，加上它们的排列方式和周围支撑结构，组成组织; 多个组织合起来成为器官**

## 2. 组织 = 一群细胞 + 它们的排列方式, 共同承担的基础功能

上皮像墙纸，肌肉像发动机，神经像电线，结缔组织像钢筋、水泥、填充层、物流系统。

#### 2.1. `上皮组织`: 人体的 贴面-覆盖层

- 皮肤最外面那层
- 口腔、食道、胃、肠道内壁那层
- 血管、器官表面的一些覆盖层

功能:

- 覆盖：把里面保护起来
- 阻隔：不让东西随便进出
- 吸收：比如肠道吸收营养
- 分泌：比如腺体分泌黏液、消化液

#### 2.2. `肌肉组织`: 人体的 动力层

- `骨骼肌`：运动, 比如: 抬手、走路、转头
- `心肌`： 心脏跳
- `平滑肌`：让 肠道、血管 收缩

#### 2.3. `神经组织`: 人体的 通信-控制层

神经元 -- 神经组织 -- 脑、脊髓、神经

组成:

- `神经元`: 传递信息
- `胶质细胞`: 支持、保护、营养、维护环境

功能: **接收信号 → 传递信号 → 处理信号 → 发出指令**

- 皮肤感觉到烫
- 信号传到神经系统
- 大脑或脊髓处理
- 再发命令让手缩回来

#### 2.4. `结缔组织`: 人体的 支撑-连接层

基建层:

- 把别的东西 **连起来、托起来、包起来、垫起来、运起来**
- 提供 结构支持、连接关系、内部环境。

组成 例子:

- 疏松结缔组织：像填充物、连接物
- `脂肪`：储能、缓冲
- `软骨`, `骨`
- `血液`, `淋巴`

## 3. 器官

例子: **心脏 = 心肌组织 + 结缔组织 + 神经组织 + 血管 + 上皮组织 等**

按系统分, 见 第二天的 系统

按 器官长相, 工作方式 分类:

##### 3.1. 实质性器官: 一整块会干活的器官

心脏 / 肝脏 / 脾脏 / 肺 / 肾脏 / 胰腺

#### 3.2. 空腔性器官: 身体的 管子、袋子、通道

里面有腔、有管道，东西会从里面通过。

胃 / 肠 / 膀胱 / 子宫 / 胆囊 / 胆管 / 血管 / 气管

#### 3.3. 管道 / 导管: 更偏“运输线路”

食管 / 输尿管 / 输卵管 / 胆管 / 血管 / 气管

## 4. 系统

见 第二天科普, 此处略

## 5. 场景: 缺氧

**系统层出问题 → 器官被迫调整 → 组织供氧改变 → 细胞代谢改变 → 整个人体出现症状**

- 诱因: 高海拔、肺功能差、贫血、循环变差
- 场景: 吸进来的氧不够 / 血把氧送出去的能力下降了
- 本质：细胞拿到的氧变少了。

联动:

- 系统层：加快呼吸、加快心跳、重分配血流
- 器官层：优先保脑和心，外围器官让资源
- 组织层：局部组织供氧下降，集体功能变差
- 细胞层：`ATP` 不足，转无氧代谢，乳酸增多，严重时细胞死亡

### 5.2. 系统层

人体不会等细胞快死了才反应; 它会先在 系统层 做紧急调整。

#### 1. 呼吸系统先加班

- 呼吸更快, 呼吸更深
- 目的: 尽量多吸氧进来
- 表现：喘 / 呼吸急促 / 觉得“吸不够气”

#### 2. 循环系统也加班

- 心血管系统会想办法把有限的氧更快送出去：
  - 心率加快
  - 心输出量增加
  - 某些部位血管收缩，优先把血给关键器官

开始做“资源倾斜”：

- 优先保 脑
- 优先保 心脏
- 尽量维持 肺、肾 等关键部位灌注
- 皮肤、四肢、消化道等部分区域可以暂时“忍一忍”

#### 3. 神经-内分泌系统 参与指挥

调度信号：

- 交感神经兴奋
- 肾上腺素等应激信号增加
- 长一点时间后，肾脏会提高 促红细胞生成素（EPO）

意思是：

- 短期先靠 “喘得更快、心跳更快”
- 中长期再想办法 “造更多红细胞，提高运氧能力”

### 5.3. 器官层

#### 1. 肺

如果问题在肺：氧进血的效率下降, 血氧变低

肺不只是“一个器官坏了”，而是把全身氧供应一起拉低。

#### 2.心脏

- 心脏：跳得更快 / 泵得更多
- 心脏为了救全身，自己也更耗氧。
- 轻度缺氧 能代偿；

* 严重缺氧时，连心肌自己都缺氧: 心慌, 胸闷, 心律失常, 泵血能力下降

#### 3.脑

脑对缺氧特别敏感，耗能高，不储备能量。

表现: 头晕, 注意力差, , 反应慢, 烦躁, 嗜睡, 意识模糊

#### 4.肾

肾脏 感到 供氧不足，长期会启动补偿：

- 分泌更多 EPO
- 促进骨髓造更多红细胞

#### 5. 消化系统、肌肉、皮肤

让资源： 消化变差 / 四肢发冷 / 容易乏力 / 活动能力下降

身体: 先保命，再保舒适；先保核心器官，再保外围。

## 5.4. 组织层

- 系统层 决定: 把多少氧送来
- 组织层 决定: 拿到这些氧之后，这一层还能不能正常集体工作

当供氧下降时，每个器官里的不同组织，会一起被影响。

心脏：

- 心肌组织：收缩需要很多 ATP，缺氧后收缩效率下降
- 神经组织：自主调节可能受影响
- 结缔组织/血管：局部灌注和支撑环境变化
- 上皮/内皮层：屏障、交换状态也会改变

脑：

- 神经组织最怕缺氧
- 血管内皮和周围支持环境也会变
- 局部供血稍差的区域会先出问题

## 5.5. 细胞层, 缺氧 = 细胞发电效率下降

- 细胞 要的是
  - 燃料: 葡萄糖 / 脂肪
  - 氧气
- 线粒体::用 氧 把 燃料 变 `ATP`
- `ATP`: 细胞的通用能量币

#### 1. 线粒体发电下降

线粒体的高效产 ATP 能力下降。

ATP 变少, 细胞“没电了”

#### 2. 细胞改用低效备用方案

细胞会更多靠 无氧糖酵解 顶着。

能量产得少得多, 乳酸增加, 局部环境更酸

缺氧 = 能量不足 + 酸中毒倾向

#### 3. 离子泵开始撑不住

细胞膜上很多“泵”要靠 ATP 工作。

- 钠离子调节变差, 水进细胞, 细胞肿胀, 钙离子稳态变差

细胞是 “精密控水控电解质的小工厂”，缺氧后先从“供电不足”变成“内部秩序混乱”。

#### 4. 功能先坏，再结构坏

缺氧先是：功能异常, 代谢紊乱, 膜、骨架、细胞器受损, 严重时细胞死亡

# 第2天: 人体的系统

## 1. 目标

- 人体的 11 个系统分别是什么
- 每个系统大概由哪些器官组成、职责是什么
- 整体视角：在一起维持人体稳定运行，也就是为`稳态`服务

## 2. 主线, 对每个系统

- 主要由哪些器官构成？
- 核心是干什么的？
- 和 哪些系统 边界清楚，和哪些系统 天然交叉？
- “吃饭 → 吸收 → 运输 → 利用 → 排废” 这条线上，处在什么位置？

## 3. 总框架：人体为什么要分“系统”来看

`系统`: 一组器官和组织，围绕某个核心任务长期协作

- 单看器官，容易碎片化
- 按系统看，才能理解 **输入—处理—分配—控制—排出** 的整体流程

人体大功能：

- 运输类：`循环系统`
- 供氧/换气类：`呼吸系统`
- 摄入/加工类：`消化系统`
- 排废/调水盐类：`泌尿系统`
- 控制类：`神经系统`、`内分泌系统`
- 防御类：`免疫系统`
- 支撑与运动类：`骨骼系统`、`肌肉系统`
- 边界与保护类：`皮肤系统`
- 延续：`生殖系统`

## 4. 概览

- 有些 系统 按 器官分的，有些更像 按功能分 的。
- **跨很多器官: 免疫、内分泌、神经**

### 4.1 `循环系统`

- 器官：心脏、血管、血液
- 职责：运输
  - 运什么：氧、营养、激素、免疫细胞、代谢废物、热量
- 关系最密切：呼吸、消化、泌尿、内分泌、免疫
- 特点：物流网, 连着 所有系统

### 4.2 `呼吸系统`

- 器官：鼻、咽、喉、气管、支气管、肺
- 职责：气体交换
  - 任务：给血液补氧，把二氧化碳排出去
- 关系最密切：循环系统
- 特点：功能上 和 循环系统 几乎绑定

### 4.3 `消化系统`

- 器官：口腔、食管、胃、小肠、大肠、肝、胆、胰、肝
- 职责：摄入、分解、吸收
- 关系最密切：循环、内分泌、免疫
- 特点：吃进去 + 营养加工

### 4.4 `泌尿系统`

- 器官：肾、输尿管、膀胱、尿道
- 职责：排废、调水、调盐、调酸碱
- 关系最密切：循环、内分泌
- 特点：排尿 + 调节 内液体化学环境

### 4.5 `神经系统`

- 器官：脑、脊髓、周围神经、感觉器官相关通路
- 职责：快速感知、判断、指挥
- 关系最密切：全系统
- 特点：高速控制网，反应快，擅长秒级调节

### 4.6 `内分泌系统`

- 器官：下丘脑、垂体、甲状腺、肾上腺、胰岛、性腺等
- 职责：激素调节
- 关系最密切：神经、消化、泌尿、生殖、循环
- 特点：慢速控制网，反应慢，影响广而持久

### 4.7 `免疫系统`

- 器官：骨髓、胸腺、脾、淋巴结、淋巴管、白细胞等
- 职责：安检、防御、清理
- 关系最密切：循环、皮肤、消化
- 特点：边界最不整齐，因为免疫细胞几乎到处跑

### 4.8 `骨骼系统`

- 器官：骨、关节、软骨、韧带
- 职责：支撑、保护、杠杆
- 关系最密切：肌肉、免疫
- 特点：骨髓参与造血

### 4.9 `肌肉系统`

- 器官：骨骼肌、平滑肌、心肌
- 职责：运动、推动、维持姿势、产热
- 关系最密切：骨骼、神经、循环
- 特点：走路 + 肠道蠕动 + 心脏跳动

### 4.10 `皮肤系统`

- 器官：皮肤、毛发、指甲、汗腺、皮脂腺
- 职责：屏障、保护、体温调节、感觉
- 关系最密切：免疫、神经、循环
- 特点：外壳 + 第一道防线

### 4.11 `生殖系统`

- 器官：男性与女性生殖器官、相关腺体
- 职责：产生配子、繁殖、性激素分泌
- 关系最密切：内分泌
- 特点：对 物种延续 必不可少

## 5. 边界

### 5.1 边界相对清楚的

- `呼吸系统`
- `消化系统`
- `泌尿系统`
- `生殖系统`
- `皮肤系统`

### 5.2 天然高度交叉的

- `神经系统` + `内分泌系统`：控制系统，快 / 慢
- `循环系统` + `呼吸系统`：换气，运输
- `循环系统` + `消化系统`：吸收完 必须 靠血来送
- `循环系统` + `泌尿系统`：肾 处理血 形成尿
- `免疫系统` + `循环系统`：免疫细胞 靠 液体 运输到全身
- `骨骼系统` + `肌肉系统`：成对理解
- `皮肤系统` + `免疫系统`：第一道屏障 + 后续防御

### 5.3 五官: 和 外界交互 的 前端接口

- 输入 信息：光、声、气味、味道、触觉
- 输入 物质：空气、食物、水
- 输出：语言、表情
- 系统: 负责处理

很大程度上:

- 眼 = 感觉器官 + 视觉神经
- 耳 = 感觉器官 + 听觉神经
- 鼻 = 感觉器官 + 嗅觉神经 + 呼吸系统
- 舌头 = 味觉器官 + 一部分体感器官
  - 化学刺激 引发 的 体感神经: 辣 (痛感), 麻, 凉 (薄荷)

## 6. "吃饭" 串 系统

“吃饭”不是消化系统单干，而是 消化、循环、呼吸、泌尿、神经、内分泌 一起完成的一条流水线。

糖 / 氨基酸: 食物 → 小肠分解吸收 → 肠绒毛毛细血管 → 门静脉 → 肝 → 全身循环 → 细胞

脂肪（简化版）: 食物脂肪 → 小肠分解吸收 → 乳糜管 → 淋巴 → 大静脉血液 → 全身循环

- 小肠 = 收货口
- 肝 = 总仓库 + 质检中心 + 分拣中心
- 循环系统 = 配送网络

### 6.1 起点：吃进去

- 食物进入口腔、胃、小肠
- 被 `消化系统` 分解成可吸收的小分子

### 6.2 吸收入血

- 小肠把营养吸收进入血液或淋巴
- `消化系统` 把“外界物质” 交给 `循环系统`

### 6.3. 肝 = 化工厂 + 中转站 + 解毒厂 + 仓库 + 调度中心

- 解毒, 处理废物
- 管营养
- 管血糖: 平衡 成分
- 造蛋白
- 造胆汁

### 6.4 运输到全身

- 心脏 和 血管 把 葡萄糖、氨基酸、脂质、维生素 送到各处
- 呼吸系统 把氧 送入 血液，供细胞使用

### 6.5 细胞使用

- 细胞 拿到 氧 和 营养，进行 代谢，产能、合成、修复

### 6.6 产生废物

- 细胞 代谢后 产生 二氧化碳、含氮废物、多余水分等

### 6.7 废物排出

- 二氧化碳 主要交给 `呼吸系统` 排出
- 尿素、多余盐分、多余水 交给 `泌尿系统` 处理
- 未吸收残渣 由 `消化系统` 继续形成 粪便 排出

### 6.8 协调

- `神经系统` 监测 状态、调节食欲、口渴、呼吸频率
- `内分泌系统` 用 激素 调节 血糖、代谢速度、储能和释放
- `免疫系统` 在 肠道 和 全身 监控异常
- 散热保温: `皮肤系统`, `循环系统`

## 7. `稳态` 简介

`稳态`: 在 波动中 维持相对 稳定

系统为`稳态`服务, 把 人体的关键指标 维持在可生存范围:

- 体温
- 血糖 / 血压 / 血氧
- 水盐平衡 / 酸碱平衡

各系统的协作目标: 把 内部环境 维持在 能活、能工作、能修复 的范围里。

# 第3天: `稳态` Homeostasis

## 1. 目标

“活着”就是 在不断波动中，把内部环境拉回可接受范围

- 身体追求的 不是“完全不变”, 是 动态平衡
- 控制系统 `稳态` = 感知 → 判断 → 调节
  - 身体 持续监测 内部环境，通过调节，把 关键指标 维持 在适合细胞生存的范围内。

关键指标:

- 体温 / 血糖 / 血压 / 血氧 / 水和盐的浓度 / 酸碱度
- 重点：控制在一个范围内, **不能飘太远、飘太久**
- 血糖、血压 随 进食、运动、睡眠变化

乱飘 后果:

- 体温 乱飘：酶活性 异常
- 血糖 乱飘：供能 不稳
- 血压 太低：器官 灌注不足
- 渗透压 乱飘：细胞 胀裂, 皱缩
- 酸碱度 乱飘：蛋白功能, 离子平衡被破坏

## 2. 关键词: 控制论 范畴

### 2.1 `反馈`（Feedback）

系统根据结果，反过来影响自己后续动作。

- 负反馈:偏离正常值时，系统启动反向调节，把它拉回来。
  - 体温高了 → 出汗、皮肤血管扩张 → 散热
  - 血糖低了 → 升糖激素启动 → 血糖回升
- 正反馈: 变化一旦开始，会继续放大
  - 不适合长期维持稳态，更多用于“快速完成某件事”。
  - 分娩时宫缩增强, 凝血过程逐步放大

### 2.2. 体温调节

- 目标：让体温维持在 适合酶 和 细胞 工作的范围。
- 太高：蛋白质和酶工作异常; 细胞代谢紊乱; 严重时组织受损
- 太低：酶活性下降; 神经和肌肉功能变差; 全身代谢速度下降

### 2.3 血糖调节

- 目标：保证细胞，尤其是脑，有稳定能量来源。
- 太高：长期会损伤血管、神经、肾、眼底等;
  - 糖进入细胞或被处理的能力出了问题
- 太低：脑供能会先出问题; 出现 头晕、出汗、心慌、意识障碍

### 2.4 酸碱平衡

- 目标：让体液 pH 保持在很窄的可用范围。
  - 酶对 pH 很敏感, 血液太酸或太碱，很多化学反应都会失调
- 神经、心脏、呼吸系统都会受影响

## 3. 器官 与 稳态

稳态是多器官协作的控制网络。

### 3.1 脑：总控与整合中心

- 下丘脑：体温、饥饿、口渴、内分泌调节的重要中枢
- 脑干：呼吸、心率、血压等基础生命活动调控

作用：接收来自全身的信号; 判断是否偏离目标区间; 发出神经和激素层面的调节命令

### 3.2 内分泌腺：激素调度系统

- 胰腺：血糖调节
- 肾上腺：应激、血压、水盐平衡
- 甲状腺：代谢速度
- 垂体：很多激素轴的总调度
- 下丘脑-垂体轴：神经系统和内分泌系统的接口

作用：用激素做“远程广播”; 让肝、肾、肌肉、脂肪等器官一起配合

### 3.3 肝：代谢与储备调节中心

- 储存 和 释放 葡萄糖
- 参与 脂质、氨基酸 代谢
- 解毒 和 处理体内外来的化学物质
- 参与 血浆 蛋白 合成

### 3.4 肾：水、盐、酸碱 的精细调节器

**肾在稳态中的地位极高**。

- 调节 水平衡
- 调节 电解质: 钠、钾
- 调节 酸碱平衡
- 参与 血压调节

### 3.5 肺：气体交换与酸碱调节的重要参与者

二氧化碳过多，会推动体液向“更酸”的方向走。

肺不只是吸氧; 还参与：

- 排出二氧化碳
- 影响 血液酸碱度
- 配合 脑, 肾 维持 pH 稳定

### 3.6 心脏和血管：运输与压力维持系统

- 保证氧气、营养、激素送达各处
- 维持足够灌注压，让脑、肾等器官能工作
- 参与体温散热和血压调节

## 4. 流程

### 4.1 `传感`: 身体先发现“偏了”。

- 温度 感受器 发现 体温升高
- 胰岛细胞 感知 血糖变化
- 血管和脑 感知 血压变化
- 化学 感受器 感知 氧气、二氧化碳、pH变化

### 4.2 `判断`: 对比 当前值 和 目标范围

- 脑，特别是 **下丘脑、脑干**
- 某些 内分泌器官 能 局部判断

### 4.3 `调节`: 发 神经, 激素 信号

- 血管, 汗腺, 肌肉, 肝, 肾, 脂肪, 内分泌腺

### 4.4 `反馈`

- 调节后再次检测，看是否回到正常区间。
- 如果还没回去：继续调 / 提高调节强度
- 调节系统坏：进入疾病状态

## 5. 例子: 体温

体温升高时，身体如何把温度拉回来

1. 异常: 外界太热、运动发热、感染发热 导致 体温上升
2. 传感: 皮肤, 中枢 的 温度感受器 传到 下丘脑
3. 下丘脑 判断：当前体温高于目标范围，需要散热
4. 调节

- 出汗增加：蒸发散热
- 皮肤血管扩张：更多热量从皮肤散出
- 减少产热行为：想休息、减少活动
- 行为层调节：找阴凉、喝水、脱衣服

5. 反馈: 体温下降后，出汗和血管扩张减弱，系统回到较平衡状态

顺便说下: 感染的发烧，不完全等于系统失控。

- 免疫系统改变了体温设定点
- 身体主动进入“更高温”的防御模式

## 6. 疾病 = 关键变量没稳住

很多疾病，不只是 某个器官 坏了，而是 稳态网络 维持不住

- 糖尿病：血糖稳态 失灵
- 高血压：血压调控 长期偏离
- 脱水 / 电解质紊乱：水盐稳态 失灵
- 肾衰竭：酸碱、水盐、代谢废物稳态 失灵
- 休克：循环稳态 崩溃，器官灌注失败
- 高热 / 低体温：体温稳态 失控
- 呼吸衰竭：氧和二氧化碳 稳态 失灵

# 第4天: 控制网络: 神经系统 vs 内分泌系统

神经信号、神经元、突触、激素、受体、反馈回路

## 1. 主线

- 神经系统：更像即时控制，快、准、局部、短时
- 内分泌系统：更像系统级调节，慢一些、广一些、持续更久

问题:

- 神经系统靠什么传消息？
- 内分泌系统靠什么传消息？
- 细胞为什么能“听懂”这些消息？

## 3. 结构差别：布线网络 vs. 血液广播网络

### 3.1 结构: 神经系统

神经信息 在单个神经元内 靠 电信号 沿细胞膜 传导；到末端 和下个细胞间 不是连续一体，在`突触`处 用 神经递质 把 消息交过去。

- `神经元`是单个细胞, 由 细胞体、树突、轴突；
  - `树突`：接收别人的信号
  - `轴突`：把自己的信号往外送
  - `轴突末端`：把消息交给 下一个细胞: 另一个神经元 / 肌肉细胞 / 腺体细胞
  - 传给 肌肉细胞：把命令变成动作
  - 传给 腺体细胞：把命令变成分泌和调节
- `神经` 是 许多`神经元` 伸出来 的 轴突-纤维 打包 形成 的 信息传输线。

### 3.2 结构: 内分泌系统

- 内分泌腺体 负责放 “激素”
- 激素 进入血液
- 血液 带到全身
- 只有 带对应受体 的 细胞会响应

##### 3.2.1 激素 = 身体内部发出的“全身调节指令”

激素 通过血液 把调节信号 送到全身，让带对应受体的细胞改变工作状态，从而调节人体的整体运行。

- 现在该不该多供能
- 现在该不该长身体
- 现在该不该储存能量
- 现在该不该进入 紧张/应激 状态
- 现在水和盐要不要多留一点

## 4. 功能差别：快系统 vs. 慢系统

### 4.1 传递速度

- 神经信号 沿 神经纤维 快速传导
- 激素要先分泌、进入血液、再运输到目标处

### 4.2 作用范围

- 神经更适合“点对点”精确控制
- 激素更适合“多器官同步调节”

### 4.3 持续时间

- 神经反应常常来得快、去得也快
- 激素起效慢一点，但能维持更久

## 5. 案例: 惊吓时身体发生了什么

- 一起维持状态: 神经先启动, 激素后跟上
- 神经负责先按下按钮, 内分泌负责把全身状态拉到位并维持住

### 5.1 第一步：感知刺激

- 眼、耳等感觉器官发现异常
- 信息送入神经系统

### 5.2 第二步：神经快速反应

- 大脑/中枢 快速判断 “有危险”
- 交感神经 迅速 启动
- 心跳加快、呼吸变急、汗腺开始活动、注意力集中

### 5.3 第三步：激素接力维持

- 肾上腺 等 内分泌器官被调动
- 激素 进入 血液
- 让身体 维持 更高警觉、更高供能状态

### 5.4 第四步：反馈与恢复

- 危险过去后，系统不能一直紧绷
- 通过 反馈回路 逐渐 把身体拉回稳态

# 第5天: 三大后勤: 供氧、供能、排废

## 1. 主线: 细胞要活着，就不断拿到 原料、用掉原料、排出废物

- 人体多个系统 在持续解决3件事：供氧, 供能/供料, 排废
- 人体的系统分工，本质上是在为海量细胞做后勤。

过程:

- 供氧: 外界空气 → 呼吸系统 → 血液 → 组织细胞 → 二氧化碳回到血液 → 肺排出
- 供能: 外界食物 → 消化系统 → 血液 → 组织细胞 → 代谢废物回到血液 → 肾等器官排出

## 2. 核心词

- 氧气：细胞 高效释放能量时 常要用到的 关键原料
- 营养：从 食物里 拆出来、能被身体利用的小分子原料
- 二氧化碳：细胞代谢 产生的 主要废物
- 代谢废物：细胞工作后产生、需要清除的各种“垃圾”

直觉:

- 细胞不是“天然就能活”, 一直在消耗输入，也一直在产生输出;
- 不断进货、不断送货、不断清垃圾，生命活动才能维持

## 3. 细胞层: 细胞为什么需要后勤

- 不是人体先有系统，细胞才需要后勤；
- 因为细胞一直需要后勤，人体才演化出这么多系统。

### 3.1 细胞为什么需要供氧

- 细胞要工作，要消耗能量
- 很多细胞为了更高效地产能，需要氧气参与

### 3.2 细胞为什么需要供能

- 细胞不能凭空制造原料
- 它需要 葡萄糖、氨基酸、脂质、电解质、水 等输入

### 3.3 细胞为什么需要排废

- 细胞工作会产生二氧化碳和其他代谢产物
- 如果垃圾堆积，局部环境会变差，细胞功能会受影响

## 4. 系统层：人体为什么需要这么多系统

呼吸系统管氧气进出，消化系统管营养进货，循环系统管全身配送和回收，泌尿系统管废物清理与环境稳定。

### 4.1 `呼吸系统`：和外界交换气体

氧 -- 二氧化碳

### 4.2 `消化系统`：把外界食物变成可吸收原料

- 食物先被拆解
- 再把能吸收的小分子送进体内

### 4.3 `循环系统`：全身运输

- 把氧和营养送到各组织细胞
- 把二氧化碳和其他废物带走

### 4.4 `泌尿系统`：负责清理血液中的部分代谢废物，调节体液环境

- 把某些废物排出体外
- 维持水、电解质、酸碱平衡

## 5. 为什么不能只靠一个系统

任务不同，结构就不同；结构不同，才会出现系统分工。

- 气体交换 要有很大的交换表面 → 肺擅长
- 食物拆解和吸收 是另一种工作 → 消化道擅长
- 长距离配送 需要流动网络 → 血液循环擅长
- 废物筛出和体液微调 需要专门过滤系统 → 肾脏擅长

## 6. 案例1: 从 吸气 到 细胞用上氧

肺不是直接把氧送给每个细胞的，真正负责“配送到全身”的是循环系统。

### 6.1 空气进入呼吸道

吸气后，空气进入鼻/口、气管、支气管，最后到肺

### 6.2 氧气到达肺泡

- 肺泡是气体交换的关键地点
- 氧气从肺泡这一侧进入血液

### 6.3 血液把氧运走

- 循环系统把带氧的血送往全身
- 氧气随着血流到达各组织

### 6.4 氧气进入组织和细胞

- 氧从血液进入组织液，再进入细胞
- 细胞利用氧进行代谢，释放能量

## 7. 案例2: 细胞 送走哦 CO2

`呼吸系统` 负责 “和外界交换”，`循环系统` 负责 “在体内搬运往返”。

### 7.1 细胞代谢产生二氧化碳

细胞工作后会产生二氧化碳

### 7.2 二氧化碳进入血液

二氧化碳从组织回到血液里

### 7.3 血液把它送回肺

循环系统把二氧化碳带回肺部

### 7.4 呼气把它排出体外

- 在肺里完成反向交换
- 二氧化碳随呼气离开身体

## 8. 补充: 营养和其他废物怎么接上这条链

把“供氧”扩展到“供能与排废”。

### 8.1 营养从哪里来

- 食物进入消化系统
- 被拆成可吸收的小分子
- 进入血液或淋巴，再被送到细胞

### 8.2 其他废物怎么处理

- 细胞不只产生二氧化碳
- 还有其他代谢废物进入血液
- 肾脏从血液中筛出一部分，最终形成尿液排出

### 8.3 总结

- 氧气：呼吸系统进，循环系统送
- 营养：消化系统进，循环系统送
- 废物：循环系统回收，肺和肾等负责排出

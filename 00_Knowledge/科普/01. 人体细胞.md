# 人体细胞

+ 细胞 → 组织 → 器官 → 系统 → 整个人体
+ 四大组织: 上皮组织, 肌肉组织, 神经组织, 结缔组织

## 0. 人体 11 个 系统

+ 皮肤系统: 皮肤 + 毛发 + 指甲 + 汗腺 + 皮脂腺 
+ 骨骼系统: 骨 + 关节 + 韧带
+ 肌肉系统: 心肌 + 骨骼肌 + 平滑肌
+ 神经系统: 脑 + 脊髓 + 周围神经
+ 内分泌系统: 垂体 + 甲状腺 + 肾上腺 + 胰岛 + 性腺 + ...
+ 循环系统: 心脏 + 血管 + 血液
+ 淋巴 / 免疫系统: 淋巴管 + 淋巴结 + 脾脏 + 胸腺 + 扁桃体 + 各类免疫细胞
+ 呼吸系统: 鼻腔 + 咽喉 + 气管 + 支气管 + 肺
+ 泌尿系统: 肾脏 + 输尿管 + 膀胱 + 尿道 + 前列腺
+ 生殖系统: 男 / 女 性生殖器官
+ 消化系统: 口腔 + 食管 + 胃 + 小/大肠 +  + 肝 + 胰 + 胆囊

特殊: 肝脏, 属于消化系统

+ 消化功能: 产生胆汁 + 脂肪消化 + 处理营养
+ 代谢 / 解毒: 药物 / 酒精 / 有毒代谢产物 / 合成蛋白 / 储存糖原

例子:

+ 怎么供氧？
    - 呼吸系统 + 循环系统
+ 怎么拿营养？
    - 消化系统 + 循环系统
+ 怎么排废物？
    - 泌尿系统 + 呼吸系统 + 消化系统 + 皮肤 + 肝脏参与代谢处理
怎么控制全身？
    - 神经系统 + 内分泌系统
+ 怎么防御敌人？
    - 免疫 / 淋巴系统 + 血液中的白细胞
+ 怎么支撑和运动？
    + 骨骼系统 + 肌肉系统 + 神经系统

## 1. 上皮组织: 外壳、接口层

细胞：上皮细胞

作用：屏障, 吸收, 分泌, 保护

位置：皮肤表面 / 肠道内壁 / 呼吸道内壁 / 腺体

## 2. 肌肉组织: 动力系统

细胞：肌细胞 / 肌纤维

分类:

+ 骨骼肌：控制手脚运动
+ 心肌：心脏跳动
+ 平滑肌：胃肠蠕动、血管收缩

作用：

+ 产生力量
+ 让身体和器官运动

## 3. 神经组织: CPU + 通信

细胞：

+ 神经元
+ 神经胶质细胞

作用：

+ 传递电信号
+ 处理信息
+ 控制运动、感觉、记忆、反应

## 4. `结缔组织`: 钢筋、水泥、物流

常见细胞：

+ 成纤维细胞：制造 胶原 等“建筑材料”
+ 脂肪细胞
+ 软骨细胞
+ 骨细胞
+ `免疫细胞`
+ `血细胞`

作用：

+ 支撑身体结构
+ 连接和填充各组织
+ 储能
+ 运输物质
+ 参与防御和修复

## 5. 血细胞

+ 血液 = 血浆 + 有形成分
+ 血浆：水、蛋白质、盐、激素、营养、废物……
+ 有形成分
    - `白细胞：免疫`
    - 红细胞：运氧
    - 血小板：止血、凝血
        - 血小板更像 细胞碎片 / 细胞片段

## 6. 免疫细胞

免疫中的白细胞:

+ `粒细胞`: 前线硬打
    - 中性粒细胞
    - 嗜酸性粒细胞
    - 嗜碱性粒细胞
+ `单核细胞`
+ `淋巴细胞`
    - `T 细胞`: 精确指挥和杀伤
    - `B 细胞`: 造抗体
    - `NK 细胞`: 快速杀异常

驻扎在 组织 的 免疫细胞

+ `巨噬细胞`: 单核细胞系, 吞噬、清理、汇报
+ `树突状细胞`
+ 肥大细胞: 过敏警报

## 7. `淋巴系统`

人体 有两套 管道: 

A. 血管: 传输 氧、营养、废物

+ 心脏泵血
+ 血液在血管里高速流动

B. 淋巴系统: 排泄 + 安检

+ 回收 组织里 多余的液体
+ 顺便 做 免疫检查
+ 带 外来物、碎片、病原体 到 检查站

淋巴液: 血浆渗出 → 组织液 → 进入淋巴管后叫淋巴液

淋巴液一路流，会经过 `淋巴结`，免疫细胞 在 `淋巴结` 检查：

+ 细菌
+ 病毒
+ 异常蛋白
+ 可疑细胞

## 8. 脾脏: 血液安检中心

会过滤血液里的：

+ 病原体
+ 老旧红细胞
+ 异常颗粒
+ 被抗体标记的东西

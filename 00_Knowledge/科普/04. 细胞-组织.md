# 第2周: 细胞与组织

# Day 6 — 细胞：生命活动的最小工作单元

## 1. 结论

- 人体所有复杂功能，最后都落到细胞层执行
- **细胞不是水袋，是 有门禁、有车间、有规则中心 的 小系统**

## 2. 结构: 细胞地图

- 细胞膜: 门禁, 识别 和 边界
- 细胞质: 车间, 传递 和 执行
- 细胞核: 规则中心, 长期规则调整

### 2.1 `细胞膜`: 控制进出

- 细胞最外层边界, 把细胞内外分开
- 表面有受体，负责识别外界信号

### 2.2 `细胞质`: 细胞的主要执行区

- 膜 和 核 之间
- 进行 化学反应
- 蛋白、`囊泡`、细胞骨架 在 工作

### 2.3 `细胞核`: 放 DNA 的区域

- 信息 和 规则 管理中心
- 影响细胞长期工作模式

## 3. 功能: 细胞不是 “装水的小袋子”

- 识别外界信号
- 控制物质进出
- 能量转换 和 物质加工
- 传递信息 / 改变 行为 / 长期调整 工作状态

## 4. 过程：胰岛素如何改变细胞行为

饭后胰岛素到来 → 细胞表面受体识别 → 信号传入细胞内部 → 细胞改变葡萄糖摄取和代谢行为

### 4.1 外界信号到来

饭后血糖升高, 胰腺分泌胰岛素, 胰岛素随血液到达细胞周围

### 4.2 细胞膜识别信号

胰岛素先接触细胞膜, **只有带胰岛素受体的细胞才能“听懂”**

### 4.3 信号传入 细胞内部

受体被激活, 外界信号被转成细胞内部消息, 细胞质中的蛋白开始接力传递

### 5.4 细胞质: 执行反应

- `囊泡`中的`运糖蛋白`被搬到`细胞膜`上
- `葡萄糖`更容易进入`细胞`
- 细胞代谢状态随之改变

### 4.5 细胞核: 参与长期调节

某些持续信号 会影响 基因表达, 细胞长期工作模式发生调整

# Day 7 — 细胞器总览：核、线粒体、核糖体、内质网、高尔基体

## 1. 目标

细胞内部不是一锅粥，而是一个分工明确的工厂。

- 为什么细胞里要有这么多“房间”和“车间”？
- 哪些细胞器负责“出指令”，哪些负责“造东西”，哪些负责“加工和发货”？
- 一个要分泌到细胞外的蛋白，是怎么一步步被造出来的？
- 溶酶体这种“回收站/拆解站”为什么也很重要？

## 2. 主线

蛋白 准备分泌到细胞外，从 DNA 指令开始，转录、翻译、加工、包装、运输，最后被释放到细胞外。

配角:

- `线粒体`：给整个流程供能
- `溶酶体`：处理废料、回收旧货、拆错货

信息处理、能量供应、生产加工、物流运输、垃圾回收

- `细胞核`：总控室 / 图纸库
- `核糖体`：装配机器
- `内质网`：生产线
- `高尔基体`：精修、贴标签、分拣、打包中心
- `线粒体`：发电站
- `溶酶体`：垃圾处理厂 / 回收站
- `囊泡`：快递箱 / 运输车

`细胞核` → `核糖体` → `粗面内质网` → `高尔基体` → `囊泡` → `细胞膜外排`

为什么要分车间:

- 避免互相干扰:
  - 不同反应需要不同环境。
  - 有的适合合成蛋白，有的适合降解废物; 如果全混一起，会互相破坏。
- 提高效率:
  - 把同类步骤放在一起，就像流水线。
  - 原料到位、酶到位、加工顺序固定，效率更高。
- 物流系统: 方便管理和运输
  - 东西做好后，可以打包、贴标签、送到指定位置。

## 3. 结构

#### 3.1 `细胞核`：指令从哪里来

- 主要放 DNA,
- `转录`：DNA → RNA
- `mRNA`：给 核糖体 看的“施工单”

#### 3.2 `核糖体`：指令 变 蛋白

- 读 `mRNA`, 按顺序把氨基酸连起来, 装配 蛋白质
- 不是存信息的地方，而是执行装配的地方。

两类:

- 游离核糖体：做 细胞内 用的 蛋白
- 附着在粗面内质网上的核糖体：做 分泌蛋白、膜蛋白、进入内膜系统 的 蛋白

#### 3.3 `内质网`：蛋白进入流水线的第一站

`粗面内质网`:

- 表面有核糖体，所以看起来“粗”
- 新合成的某些蛋白会直接进入这里
- 在这里开始折叠、初步加工、质量检查

`光面内质网`:

- 没有核糖体
- 更多参与脂质合成、解毒、钙离子储存等

#### 3.4 `高尔基体`：精修 + 贴地址 + 发货

内质网 = 初加工车间，高尔基体 = 精加工 + 物流分拣

- 从内质网送来的蛋白到这里继续加工
- 可能会被修饰，比如 加糖链
- 然后 分类、打包、送往不同目的地

#### 3.5 `线粒体`：整个流程的供能站

- 细胞很多工作 要 能量, 没有能量，转录、翻译、运输、囊泡移动都做不好
- 线粒体 产生 `ATP`

#### 3.6 `溶酶体`：拆解和回收

细胞不是只会造东西，还要会清理、回收、报废。

- 里面有降解酶, 用来 拆掉废旧蛋白、坏掉的细胞器、吞进来的异物
- 回收后的小分子还可以再利用

# 4. 过程: 分泌蛋白 怎么出厂

#### 4.1 细胞核里 选中 某基因

图纸不直接出库，而是先复印一份施工单。

+ 细胞 需要 某种分泌蛋白，比如 某种激素、消化酶、信号分子。
+ 启动对应基因
+ `DNA` 上这段信息被读取, 转录成 `mRNA`
+ `mRNA` 从细胞核出去，进入细胞质

#### 4.2 核糖体开始翻译

蛋白不是一下子变出来的，而是按顺序装配出来的。

+ mRNA 到了核糖体这里。
+ 核糖体读取上面的密码，把氨基酸接起来。

如果这个蛋白属于“要分泌出去”的类型，翻译过程会引导核糖体附着到粗面内质网。

#### 4.3 进入粗面内质网，开始初加工

不是“做出来就完事”，而是要先过质检。

+ 折叠 成立体结构
+ 接受 初步修饰
+ 接受 质量检查: 折叠不对，可能会被留下处理，甚至送去降解。

#### 4.4 `囊泡`把货 从 内质网 送到 高尔基体

+ 蛋白被装进小囊泡中运输。
+ 囊泡 像 包装箱
+ 细胞骨架 像 运输轨道
+ 马达蛋白 像 搬运车

#### 4.5 `高尔基体`: 精加工 + 分拣

+ 进一步修饰, 贴上 “去哪”的标签
+ 打包进新的囊泡

目的地: 分泌 到 细胞外 / 细胞膜上 / 送往溶酶体 / 留在细胞内某处

#### 4.6 外排 运输到细胞膜并释放

蛋白离开细胞，不是“穿墙过去”，而是靠囊泡和膜融合释放。

+ 分泌蛋白: 囊泡 移动到细胞膜附近，和 细胞膜融合。
+ 囊泡里的内容物 被释放 到 细胞外。

# 5. 配角

`线粒体`: 底层支持

+ 基因表达要能量
+ 蛋白合成要能量
+ 囊泡运输要能量
+ 膜融合要能量

`溶酶体`: 善后 + 质控

+ 错误蛋白、老旧蛋白 可以被拆解
+ 旧细胞器 可以被回收
+ 外来颗粒被吞入后也可能送来分解

工厂里的废品处理、旧设备拆解、原料回收。

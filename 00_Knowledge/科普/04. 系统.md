# 人体 11 个系统速览

## 1. 目标

- 人体的 11 个系统分别是什么
- 每个系统大概由哪些器官组成、职责是什么
- 整体视角：在一起维持人体稳定运行，也就是为`稳态`服务

## 2. 主线, 对每个系统

- 主要由哪些器官构成？
- 核心是干什么的？
- 和 哪些系统 边界清楚，和哪些系统 天然交叉？
- “吃饭 → 吸收 → 运输 → 利用 → 排废” 这条线上，处在什么位置？

# 3. 总框架：人体为什么要分“系统”来看

`系统`: 一组器官和组织，围绕某个核心任务长期协作

- 单看器官，容易碎片化
- 按系统看，才能理解 **输入—处理—分配—控制—排出** 的整体流程

人体大功能：

- 运输类：`循环系统`
- 供氧/换气类：`呼吸系统`
- 摄入/加工类：`消化系统`
- 排废/调水盐类：`泌尿系统`
- 控制类：`神经系统`、`内分泌系统`
- 防御类：`免疫系统`
- 支撑与运动类：`骨骼系统`、`肌肉系统`
- 边界与保护类：`皮肤系统`
- 延续：`生殖系统`

# 4. 概览

+ 有些 系统 按 器官分的，有些更像 按功能分 的。
+ **跨很多器官: 免疫、内分泌、神经**

## 4.1 `循环系统`

- 器官：心脏、血管、血液
- 职责：运输
  - 运什么：氧、营养、激素、免疫细胞、代谢废物、热量
- 关系最密切：呼吸、消化、泌尿、内分泌、免疫
- 特点：物流网, 连着 所有系统

## 4.2 `呼吸系统`

- 器官：鼻、咽、喉、气管、支气管、肺
- 职责：气体交换
  - 任务：给血液补氧，把二氧化碳排出去
- 关系最密切：循环系统
- 特点：功能上 和 循环系统 几乎绑定

## 4.3 `消化系统`

- 器官：口腔、食管、胃、小肠、大肠、肝、胆、胰、肝
- 职责：摄入、分解、吸收
- 关系最密切：循环、内分泌、免疫
- 特点：吃进去 + 营养加工

## 4.4 `泌尿系统`

- 器官：肾、输尿管、膀胱、尿道
- 职责：排废、调水、调盐、调酸碱
- 关系最密切：循环、内分泌
- 特点：排尿 + 调节 内液体化学环境

## 4.5 `神经系统`

- 器官：脑、脊髓、周围神经、感觉器官相关通路
- 职责：快速感知、判断、指挥
- 关系最密切：全系统
- 特点：高速控制网，反应快，擅长秒级调节

## 4.6 `内分泌系统`

- 器官：下丘脑、垂体、甲状腺、肾上腺、胰岛、性腺等
- 职责：激素调节
- 关系最密切：神经、消化、泌尿、生殖、循环
- 特点：慢速控制网，反应慢，影响广而持久

## 4.7 `免疫系统`

- 器官：骨髓、胸腺、脾、淋巴结、淋巴管、白细胞等
- 职责：安检、防御、清理
- 关系最密切：循环、皮肤、消化
- 特点：边界最不整齐，因为免疫细胞几乎到处跑

## 4.8 `骨骼系统`

- 器官：骨、关节、软骨、韧带
- 职责：支撑、保护、杠杆
- 关系最密切：肌肉、免疫
- 特点：骨髓参与造血

## 4.9 `肌肉系统`

- 器官：骨骼肌、平滑肌、心肌
- 职责：运动、推动、维持姿势、产热
- 关系最密切：骨骼、神经、循环
- 特点：走路 + 肠道蠕动 + 心脏跳动

## 4.10 `皮肤系统`

- 器官：皮肤、毛发、指甲、汗腺、皮脂腺
- 职责：屏障、保护、体温调节、感觉
- 关系最密切：免疫、神经、循环
- 特点：外壳 + 第一道防线

## 4.11 `生殖系统`

- 器官：男性与女性生殖器官、相关腺体
- 职责：产生配子、繁殖、性激素分泌
- 关系最密切：内分泌
- 特点：对 物种延续 必不可少

# 5. 边界

## 5.1 边界相对清楚的

- `呼吸系统`
- `消化系统`
- `泌尿系统`
- `生殖系统`
- `皮肤系统`

## 5.2 天然高度交叉的

- `神经系统` + `内分泌系统`：控制系统，快 / 慢
- `循环系统` + `呼吸系统`：换气，运输
- `循环系统` + `消化系统`：吸收完 必须 靠血来送
- `循环系统` + `泌尿系统`：肾 处理血 形成尿
- `免疫系统` + `循环系统`：免疫细胞 靠 液体 运输到全身
- `骨骼系统` + `肌肉系统`：成对理解
- `皮肤系统` + `免疫系统`：第一道屏障 + 后续防御

## 5.3 五官: 和 外界交互 的 前端接口

+ 输入 信息：光、声、气味、味道、触觉
+ 输入 物质：空气、食物、水
+ 输出：语言、表情
+ 系统: 负责处理

很大程度上:

+ 眼 = 感觉器官 + 视觉神经
+ 耳 = 感觉器官 + 听觉神经
+ 鼻 = 感觉器官 + 嗅觉神经 + 呼吸系统
+ 舌头 = 味觉器官 + 一部分体感器官
  - 化学刺激 引发 的 体感神经: 辣 (痛感), 麻, 凉 (薄荷)

# 6. "吃饭" 串 系统

“吃饭”不是消化系统单干，而是 消化、循环、呼吸、泌尿、神经、内分泌 一起完成的一条流水线。

糖 / 氨基酸: 食物 → 小肠分解吸收 → 肠绒毛毛细血管 → 门静脉 → 肝 → 全身循环 → 细胞

脂肪（简化版）:  食物脂肪 → 小肠分解吸收 → 乳糜管 → 淋巴 → 大静脉血液 → 全身循环

+ 小肠 = 收货口
+ 肝 = 总仓库 + 质检中心 + 分拣中心
+ 循环系统 = 配送网络

## 6.1 起点：吃进去

- 食物进入口腔、胃、小肠
- 被 `消化系统` 分解成可吸收的小分子

## 6.2 吸收入血

- 小肠把营养吸收进入血液或淋巴
- `消化系统` 把“外界物质” 交给 `循环系统`

## 6.3. 肝 = 化工厂 + 中转站 + 解毒厂 + 仓库 + 调度中心

+ 解毒, 处理废物
+ 管营养
+ 管血糖: 平衡 成分
+ 造蛋白
+ 造胆汁

## 6.4 运输到全身

- 心脏 和 血管 把 葡萄糖、氨基酸、脂质、维生素 送到各处
- 呼吸系统 把氧 送入 血液，供细胞使用

## 6.5 细胞使用

- 细胞 拿到 氧 和 营养，进行 代谢，产能、合成、修复

## 6.6 产生废物

- 细胞 代谢后 产生 二氧化碳、含氮废物、多余水分等

## 6.7 废物排出

- 二氧化碳 主要交给 `呼吸系统` 排出
- 尿素、多余盐分、多余水 交给 `泌尿系统` 处理
- 未吸收残渣 由 `消化系统` 继续形成 粪便 排出

## 6.8 协调

- `神经系统` 监测 状态、调节食欲、口渴、呼吸频率
- `内分泌系统` 用 激素 调节 血糖、代谢速度、储能和释放
- `免疫系统` 在 肠道 和 全身 监控异常
- 散热保温: `皮肤系统`, `循环系统`

# 7. `稳态` 简介

`稳态`: 在 波动中 维持相对 稳定

系统为`稳态`服务, 把 人体的关键指标 维持在可生存范围:

- 体温
- 血糖 / 血压 / 血氧
- 水盐平衡 / 酸碱平衡

各系统的协作目标: 把 内部环境 维持在 能活、能工作、能修复 的范围里。

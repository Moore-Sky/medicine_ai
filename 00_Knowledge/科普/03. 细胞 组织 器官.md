# 人体: 细胞-组织-器官

## 1. 概述

- **细胞 → 组织 → 器官 → 系统 → 人体**
- **通常多个相似细胞，加上它们的排列方式和周围支撑结构，组成组织; 多个组织合起来成为器官**

## 2. 组织 = 一群细胞 + 它们的排列方式, 共同承担的基础功能

上皮像墙纸，肌肉像发动机，神经像电线，结缔组织像钢筋、水泥、填充层、物流系统。

#### 2.1. `上皮组织`: 人体的 贴面-覆盖层

- 皮肤最外面那层
- 口腔、食道、胃、肠道内壁那层
- 血管、器官表面的一些覆盖层

功能:

- 覆盖：把里面保护起来
- 阻隔：不让东西随便进出
- 吸收：比如肠道吸收营养
- 分泌：比如腺体分泌黏液、消化液

#### 2.2. `肌肉组织`: 人体的 动力层

- `骨骼肌`：运动, 比如: 抬手、走路、转头
- `心肌`： 心脏跳
- `平滑肌`：让 肠道、血管 收缩

#### 2.3. `神经组织`: 人体的 通信-控制层

神经元 -- 神经组织 -- 脑、脊髓、神经

组成:

- `神经元`: 传递信息
- `胶质细胞`: 支持、保护、营养、维护环境

功能: **接收信号 → 传递信号 → 处理信号 → 发出指令**

- 皮肤感觉到烫
- 信号传到神经系统
- 大脑或脊髓处理
- 再发命令让手缩回来

#### 2.4. `结缔组织`: 人体的 支撑-连接层

基建层:

- 把别的东西 **连起来、托起来、包起来、垫起来、运起来**
- 提供 结构支持、连接关系、内部环境。

组成 例子:

- 疏松结缔组织：像填充物、连接物
- `脂肪`：储能、缓冲
- `软骨`, `骨`
- `血液`, `淋巴`

## 3. 器官

例子: **心脏 = 心肌组织 + 结缔组织 + 神经组织 + 血管 + 上皮组织 等**

按系统分, 见 第二天的 系统

按 器官长相, 工作方式 分类:

##### 3.1. 实质性器官: 一整块会干活的器官

心脏 / 肝脏 / 脾脏 / 肺 / 肾脏 / 胰腺

#### 3.2. 空腔性器官: 身体的 管子、袋子、通道

里面有腔、有管道，东西会从里面通过。

胃 / 肠 / 膀胱 / 子宫 / 胆囊 / 胆管 / 血管 / 气管

#### 3.3. 管道 / 导管: 更偏“运输线路”

食管 / 输尿管 / 输卵管 / 胆管 / 血管 / 气管

## 4. 系统

见 第二天科普, 此处略

## 5. 场景: 缺氧

**系统层出问题 → 器官被迫调整 → 组织供氧改变 → 细胞代谢改变 → 整个人体出现症状**

- 诱因: 高海拔、肺功能差、贫血、循环变差
- 场景: 吸进来的氧不够 / 血把氧送出去的能力下降了
- 本质：细胞拿到的氧变少了。

联动:

- 系统层：加快呼吸、加快心跳、重分配血流
- 器官层：优先保脑和心，外围器官让资源
- 组织层：局部组织供氧下降，集体功能变差
- 细胞层：`ATP` 不足，转无氧代谢，乳酸增多，严重时细胞死亡

### 5.2. 系统层

人体不会等细胞快死了才反应; 它会先在 系统层 做紧急调整。

#### 1. 呼吸系统先加班

- 呼吸更快, 呼吸更深
- 目的: 尽量多吸氧进来
- 表现：喘 / 呼吸急促 / 觉得“吸不够气”

#### 2. 循环系统也加班

+ 心血管系统会想办法把有限的氧更快送出去：
    - 心率加快
    - 心输出量增加
    - 某些部位血管收缩，优先把血给关键器官

开始做“资源倾斜”：

+ 优先保 脑
+ 优先保 心脏
+ 尽量维持 肺、肾 等关键部位灌注
+ 皮肤、四肢、消化道等部分区域可以暂时“忍一忍”

#### 3. 神经-内分泌系统 参与指挥

调度信号：

+ 交感神经兴奋
+ 肾上腺素等应激信号增加
+ 长一点时间后，肾脏会提高 促红细胞生成素（EPO）

意思是：

+ 短期先靠 “喘得更快、心跳更快”
+ 中长期再想办法 “造更多红细胞，提高运氧能力”

### 5.3. 器官层

#### 1. 肺

如果问题在肺：氧进血的效率下降, 血氧变低

肺不只是“一个器官坏了”，而是把全身氧供应一起拉低。

#### 2.心脏

+ 心脏：跳得更快 / 泵得更多
+ 心脏为了救全身，自己也更耗氧。
+ 轻度缺氧 能代偿；
- 严重缺氧时，连心肌自己都缺氧: 心慌, 胸闷, 心律失常, 泵血能力下降

#### 3.脑

脑对缺氧特别敏感，耗能高，不储备能量。

表现: 头晕, 注意力差, , 反应慢, 烦躁, 嗜睡, 意识模糊

#### 4.肾

肾脏 感到 供氧不足，长期会启动补偿：

+ 分泌更多 EPO
+ 促进骨髓造更多红细胞

#### 5. 消化系统、肌肉、皮肤

让资源： 消化变差 / 四肢发冷 / 容易乏力 / 活动能力下降

身体: 先保命，再保舒适；先保核心器官，再保外围。

## 5.4. 组织层

+ 系统层 决定: 把多少氧送来
+ 组织层 决定: 拿到这些氧之后，这一层还能不能正常集体工作

当供氧下降时，每个器官里的不同组织，会一起被影响。

心脏：

+ 心肌组织：收缩需要很多 ATP，缺氧后收缩效率下降
+ 神经组织：自主调节可能受影响
+ 结缔组织/血管：局部灌注和支撑环境变化
+ 上皮/内皮层：屏障、交换状态也会改变

脑：

+ 神经组织最怕缺氧
+ 血管内皮和周围支持环境也会变
+ 局部供血稍差的区域会先出问题

## 5.5. 细胞层, 缺氧 = 细胞发电效率下降

+ 细胞 要的是
    - 燃料: 葡萄糖 / 脂肪
    - 氧气
+ 线粒体::用 氧 把 燃料 变 `ATP`
+ `ATP`: 细胞的通用能量币

#### 1. 线粒体发电下降

线粒体的高效产 ATP 能力下降。

ATP 变少, 细胞“没电了”

#### 2. 细胞改用低效备用方案

细胞会更多靠 无氧糖酵解 顶着。

能量产得少得多, 乳酸增加, 局部环境更酸

缺氧 = 能量不足 + 酸中毒倾向

#### 3. 离子泵开始撑不住

细胞膜上很多“泵”要靠 ATP 工作。

+ 钠离子调节变差, 水进细胞, 细胞肿胀, 钙离子稳态变差

细胞是 “精密控水控电解质的小工厂”，缺氧后先从“供电不足”变成“内部秩序混乱”。

#### 4. 功能先坏，再结构坏

缺氧先是：功能异常, 代谢紊乱, 膜、骨架、细胞器受损, 严重时细胞死亡

# `稳态` Homeostasis

## 1. 目标

“活着”就是 在不断波动中，把内部环境拉回可接受范围

- 身体追求的 不是“完全不变”, 是 动态平衡
- 控制系统 `稳态` = 感知 → 判断 → 调节
  - 身体 持续监测 内部环境，通过调节，把 关键指标 维持 在适合细胞生存的范围内。

关键指标:

- 体温 / 血糖 / 血压 / 血氧 / 水和盐的浓度 / 酸碱度
- 重点：控制在一个范围内, **不能飘太远、飘太久**
- 血糖、血压 随 进食、运动、睡眠变化

乱飘 后果:

- 体温 乱飘：酶活性 异常
- 血糖 乱飘：供能 不稳
- 血压 太低：器官 灌注不足
- 渗透压 乱飘：细胞 胀裂, 皱缩
- 酸碱度 乱飘：蛋白功能, 离子平衡被破坏

# 2. 关键词: 控制论 范畴

## 2.1 `反馈`（Feedback）

系统根据结果，反过来影响自己后续动作。

- 负反馈:偏离正常值时，系统启动反向调节，把它拉回来。
  - 体温高了 → 出汗、皮肤血管扩张 → 散热
  - 血糖低了 → 升糖激素启动 → 血糖回升
- 正反馈: 变化一旦开始，会继续放大
  - 不适合长期维持稳态，更多用于“快速完成某件事”。
  - 分娩时宫缩增强, 凝血过程逐步放大

## 2.2. 体温调节

- 目标：让体温维持在 适合酶 和 细胞 工作的范围。
- 太高：蛋白质和酶工作异常; 细胞代谢紊乱; 严重时组织受损
- 太低：酶活性下降; 神经和肌肉功能变差; 全身代谢速度下降

## 2.3 血糖调节

- 目标：保证细胞，尤其是脑，有稳定能量来源。
- 太高：长期会损伤血管、神经、肾、眼底等;
  - 糖进入细胞或被处理的能力出了问题
- 太低：脑供能会先出问题; 出现 头晕、出汗、心慌、意识障碍

## 2.4 酸碱平衡

- 目标：让体液 pH 保持在很窄的可用范围。
  - 酶对 pH 很敏感, 血液太酸或太碱，很多化学反应都会失调
- 神经、心脏、呼吸系统都会受影响

# 3. 器官 与 稳态

稳态是多器官协作的控制网络。

## 3.1 脑：总控与整合中心

- 下丘脑：体温、饥饿、口渴、内分泌调节的重要中枢
- 脑干：呼吸、心率、血压等基础生命活动调控

作用：接收来自全身的信号; 判断是否偏离目标区间; 发出神经和激素层面的调节命令

## 3.2 内分泌腺：激素调度系统

- 胰腺：血糖调节
- 肾上腺：应激、血压、水盐平衡
- 甲状腺：代谢速度
- 垂体：很多激素轴的总调度
- 下丘脑-垂体轴：神经系统和内分泌系统的接口

作用：用激素做“远程广播”; 让肝、肾、肌肉、脂肪等器官一起配合

## 3.3 肝：代谢与储备调节中心

- 储存 和 释放 葡萄糖
- 参与 脂质、氨基酸 代谢
- 解毒 和 处理体内外来的化学物质
- 参与 血浆 蛋白 合成

## 3.4 肾：水、盐、酸碱 的精细调节器

**肾在稳态中的地位极高**。

- 调节 水平衡
- 调节 电解质: 钠、钾
- 调节 酸碱平衡
- 参与 血压调节

## 3.5 肺：气体交换与酸碱调节的重要参与者

二氧化碳过多，会推动体液向“更酸”的方向走。

肺不只是吸氧; 还参与：

- 排出二氧化碳
- 影响 血液酸碱度
- 配合 脑, 肾 维持 pH 稳定

## 3.6 心脏和血管：运输与压力维持系统

- 保证氧气、营养、激素送达各处
- 维持足够灌注压，让脑、肾等器官能工作
- 参与体温散热和血压调节

# 4. 流程

## 4.1 `传感`: 身体先发现“偏了”。

- 温度 感受器 发现 体温升高
- 胰岛细胞 感知 血糖变化
- 血管和脑 感知 血压变化
- 化学 感受器 感知 氧气、二氧化碳、pH变化

## 4.2 `判断`: 对比 当前值 和 目标范围

- 脑，特别是 **下丘脑、脑干**
- 某些 内分泌器官 能 局部判断

## 4.3 `调节`: 发 神经, 激素 信号

- 血管, 汗腺, 肌肉, 肝, 肾, 脂肪, 内分泌腺

## 4.4 `反馈`

- 调节后再次检测，看是否回到正常区间。
- 如果还没回去：继续调 / 提高调节强度
- 调节系统坏：进入疾病状态

# 5. 例子: 体温

体温升高时，身体如何把温度拉回来

1. 异常: 外界太热、运动发热、感染发热 导致 体温上升
2. 传感: 皮肤, 中枢 的 温度感受器 传到 下丘脑
3. 下丘脑 判断：当前体温高于目标范围，需要散热
4. 调节

- 出汗增加：蒸发散热
- 皮肤血管扩张：更多热量从皮肤散出
- 减少产热行为：想休息、减少活动
- 行为层调节：找阴凉、喝水、脱衣服

5. 反馈: 体温下降后，出汗和血管扩张减弱，系统回到较平衡状态

顺便说下: 感染的发烧，不完全等于系统失控。

- 免疫系统改变了体温设定点
- 身体主动进入“更高温”的防御模式

## 6. 疾病 = 关键变量没稳住

很多疾病，不只是 某个器官 坏了，而是 稳态网络 维持不住

- 糖尿病：血糖稳态 失灵
- 高血压：血压调控 长期偏离
- 脱水 / 电解质紊乱：水盐稳态 失灵
- 肾衰竭：酸碱、水盐、代谢废物稳态 失灵
- 休克：循环稳态 崩溃，器官灌注失败
- 高热 / 低体温：体温稳态 失控
- 呼吸衰竭：氧和二氧化碳 稳态 失灵

# 生命周期

## 1. T细胞

#### `出生`: `骨髓` 的 造血干细胞

- 过程：造血干细胞 → 淋巴系祖细胞 → T细胞前体

#### `培训`: 胸腺 (Thymus), 所以叫 T-细胞

- 正选择: 认识 自家的 `TCR` + `MHC`
- 负选择: 淘汰 乱杀自己的 T-细胞
- 培训结果: 成熟 初始T细胞（naive T cell）
    - `CD4`：偏指挥
    - `CD8`：偏杀伤

#### `上岗`: 离开胸腺，进入：

+ 血液
+ 淋巴液
+ 淋巴结
+ 脾脏
+ 外周淋巴组织

#### `激活`:

+ 找敌人: 平时在不断巡逻： 血液 / 淋巴系统 / 淋巴结
+ 树突状细胞 / 巨噬细胞 先 发现敌人
    - 抓到 病原体 / 肿瘤相关 抗原。
+ 展示 敌人 “样本”
    - 把抗原片段放到 `MHC` 上, 就像 把嫌疑人的照片贴到公告板上。
+ T细胞 读 公告板
    - 某个T细胞 的 `TCR` 认得 这个 `抗原-MHC`，它就被激活。

#### `克隆扩增`:

T-细胞 被激活后: 

+ 大量增殖, 分化成功能不同的部队
+ 原本很少的那一小撮“正好认得这个敌人”的T细胞，会迅速复制很多份。

#### `执行` = 杀伤，调度，限流

A. `CD8`：杀伤

识别 被感染细胞 或 异常细胞表面的 抗原 后，诱导对方死亡。

+ 打孔
+ 送入致死信号
+ 触发对方自毁程序

B. `CD4`：指挥, 协调

+ 分泌 细胞因子
+ 帮助 B细胞
+ 帮助 CD8 T细胞
+ 协调 巨噬细胞
+ 调整 免疫反应强弱

C. `Treg`：踩刹车, 防止免疫系统打过头。

+ 抑制过强反应
+ 防止自身免疫
+ 帮助战后收尾

#### `回收`: 进入凋亡 / 被清除掉

+ 大多数战时扩编部队会裁军。

#### `记忆`:

+ 少部分 T细胞 变成：`记忆T细胞`, 长期存活
+ 老兵预备役 + 敌情数据库缓存。

## 2. B细胞

和 T细胞的 区别：

+ 培训地: B-骨髓, T-胸腺
+ 作用: B-造抗体, T-指挥/杀敌

#### `出生`: `骨髓` 的 造血干细胞

过程: 造血干细胞 → 淋巴系祖细胞 → B细胞前体 → 未成熟B细胞

#### `培训`: 骨髓

+ 给每个 B 细胞 装 `BCR`（B细胞受体）
+ 淘汰明显会打自己人的

#### `上岗`: 进入

+ 血液
+ 淋巴结
+ 脾脏
+ 其他外周淋巴组织

#### `激活`

+ B细胞先抓住抗原, 自己处理一下
+ 再把抗原片段展示给 CD4 看
+ T细胞确认“这个目标值得打”, 再给它激活信号

#### `扩增` & `分化`

第一部分: 变 浆细胞: 抗体工厂

+ B细胞 = 识别和决策单位
+ 浆细胞 = 抗体工厂, 疯狂产抗体，释放到血液和体液里。

`抗体`作用:

+ 绑定敌人: 先把目标标记出来。
+ 阻断敌人: 堵住病毒, 细胞结合的位置。
+ 信号: 招呼别的免疫系统来打

第二条路：变成记忆B细胞

+ 记忆B细胞: 平时不一定疯狂产抗体，但会长期活着。
+ 下次同样敌人再来时： 反应更快 / 扩增更快 / 更容易迅速产出高质量抗体

#### `回收`

+ 大量短命效应B细胞 / 浆细胞会消失
+ 少数长期浆细胞和记忆B细胞会保留下来
